<p>That's because saveVersion.sh can not execute command</p>

<pre><code>mkdir -p $src_dir/gen/org/apache/hive/common
cat &lt;&lt; EOF | \
  sed -e "s/VERSION/$version/" -e "s/SHORTVERSION/$shortversion/" \
      -e "s/USER/$user/" -e "s/DATE/$date/" \
      -e "s|URL|$url|" -e "s/REV/$revision/" \
      -e "s|BRANCH|$branch|" -e "s/SRCCHECKSUM/$srcChecksum/" \
      &gt; $src_dir/gen/org/apache/hive/common/package-info.java
</code></pre>

<p>I Modify to</p>

<pre><code>mkdir -p '$src_dir/gen/org/apache/hive/common'
cat &lt;&lt; EOF | \
  sed -e "s/VERSION/$version/" -e "s/SHORTVERSION/$shortversion/" \
      -e "s/USER/$user/" -e "s/DATE/$date/" \
      -e "s|URL|$url|" -e "s/REV/$revision/" \
      -e "s|BRANCH|$branch|" -e "s/SRCCHECKSUM/$srcChecksum/" \
      &gt; '$src_dir/gen/org/apache/hive/common/package-info.java'
</code></pre>

<p>Then resolve this problem.</p>
