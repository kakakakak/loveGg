<p>i am using hadoop with flume , hive and db derby i have twitter data in to hadoop in json formate
working fine now i need combine data as per user tweets like </p>

<pre><code>{user_id:12,text:"abcd"}
{user_id:13,text:"dcba"}
{user_id:12,text:"efjh"}
{user_id:13,text:"hjfe"}

<pre><code>{user_id:12 , tweets:["abcd","efjh"]}
{user_id:13 , tweets:["dcba","hjfe"]}

<p>i have found solution for this question:</p>

<p>hive having a list of function that modified your query result in the above question case i have found a hive function that meet the required output of question.</p>

<p>function name is collect_set().</p>

<p>usage example of this function:</p>

<pre><code>SELECT user_id , collect_set(text) FROM tweets GROUP BY user_id;
</code></pre>

<p>Result:</p>

<pre><code>  user_id        text
