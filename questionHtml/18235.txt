    <div class="post-text" itemprop="text">
<p>A quick google search returned me <a href="http://svn.apache.org/repos/asf/hadoop/core/trunk/src/core/org/apache/hadoop/util/StringUtils.java" rel="nofollow">this</a> from Appache hadoop project. Copying from there:
(Apache License, Version 2.0):</p>

<pre><code>private static DecimalFormat oneDecimal = new DecimalFormat("0.0");

  /**
   * Given an integer, return a string that is in an approximate, but human 
   * readable format. 
   * It uses the bases 'k', 'm', and 'g' for 1024, 1024**2, and 1024**3.
   * @param number the number to format
   * @return a human readable form of the integer
   */
  public static String humanReadableInt(long number) {
    long absNumber = Math.abs(number);
    double result = number;
    String suffix = "";
    if (absNumber &lt; 1024) {
      // nothing
    } else if (absNumber &lt; 1024 * 1024) {
      result = number / 1024.0;
      suffix = "k";
    } else if (absNumber &lt; 1024 * 1024 * 1024) {
      result = number / (1024.0 * 1024);
      suffix = "m";
    } else {
      result = number / (1024.0 * 1024 * 1024);
      suffix = "g";
    }
    return oneDecimal.format(result) + suffix;
  }
</code></pre>

<p>It uses 1K = 1024, but you can adapt this if you prefer. You also need to handle the &lt;1024 case with a different DecimalFormat.</p>

<td class="answercell">
    <div class="post-text" itemprop="text">
<p>You can use the commons-io <code>FileUtils.byteCountToDisplaySize</code> methods. For a JSTL implementation you can add the following taglib function while having commons-io on your classpath:</p>

<pre class="lang-xml prettyprint-override"><code>&lt;function&gt;
  &lt;name&gt;fileSize&lt;/name&gt;
  &lt;function-class&gt;org.apache.commons.io.FileUtils&lt;/function-class&gt;
  &lt;function-signature&gt;String byteCountToDisplaySize(long)&lt;/function-signature&gt;
&lt;/function&gt;
</code></pre>

<p>Now in your JSP you can do:</p>

<pre class="lang-xml prettyprint-override"><code>&lt;%@ taglib uri="/WEB-INF/FileSizeFormatter.tld" prefix="sz"%&gt;
Some Size: ${sz:fileSize(1024)} &lt;!-- 1 K --&gt;
Some Size: ${sz:fileSize(10485760)} &lt;!-- 10 MB --&gt;
</code></pre>
