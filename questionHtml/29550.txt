<td class="answercell">
    <div class="post-text" itemprop="text">
<p>My guess without seeing your source code is that you have your Map1 class defined as an inner class of WorkGraph, rather than a static class. Hadoop needs to be able to create an instance of your map / reduce classes using reflection, and require a default constructor for your class.</p>

<p>If your code looks like the following block, then your Map1 class is actually a inner child of the parent WorkGraph class, and requires a reference to the parent class to be passed in as the argument at construction (the compile hides all this from you):</p>

<pre><code>public class WorkGraph {
  public class Map1 extends Mapper {

  }
}
</code></pre>

<p>This should actually read:</p>

<pre><code>public class WorkGraph {
  public static class Map1 extends Mapper {

  }
}
</code></pre>

<p>You may find it useful to run the javap utility on your Map1 class file, as this should show you what the compiler has generated.</p>
