<p>I've been attempting to use the following commands to export and then re-import HBase table data from one server into another.</p>

<em>export</em>

<pre><code>$ sudo hbase org.apache.hadoop.hbase.mapreduce.Export -fs file:/// 'users' /tmp/users
</code></pre>

<em>import</em>

<pre><code>$ sudo -u hdfs hadoop fs -put /tmp/users /tmp/users
$ sudo -u hdfs hbase org.apache.hadoop.hbase.mapreduce.Import users /tmp/users
</code></pre>

<p>The above works initially, but then if I attempt to do the following in an <code>hbase shell</code>:</p>

<pre><code>&gt; disable 'users'
&gt; drop_all 'users'
</code></pre>
