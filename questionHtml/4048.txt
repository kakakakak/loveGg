<td class="answercell">
    <div class="post-text" itemprop="text">
<p>I too was facing the same issue on <strong><code>my Ubuntu 14.4 VitualBox</code></strong>. Here are the steps I have followed to fix:</p>

<ol>
<li><p><strong><code>hive&gt; set spark.home=/usr/local/spark;</code></strong></p></li>
<li><p><strong><code>hive&gt; set spark.master=local;</code></strong></p></li>
<li><p><strong><code>hive&gt; SET hive.execution.engine=spark;</code></strong></p></li>
<li><p>Added <strong><code>spark-assembly jar</code></strong> file as shown below:</p>

<p><strong><code>hive&gt; ADD jar /usr/local/spark/lib/spark-assembly-1.4.0-hadoop2.6.0.jar;</code></strong></p></li>
</ol>
