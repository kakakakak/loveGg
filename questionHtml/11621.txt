<td class="answercell">
    <div class="post-text" itemprop="text">
<p>You need to get permissions changed.  You have write permission to the <code>.hg/store/...</code> files that already exist, which is why you can change that file, but probably not write permissions to the <code>.hg/store</code> directory itself so you can't create new files.</p>

<p>You need to make sure your user or group on that server has rwx access to all the files and directories from the repository directory on down.  If it's group access that's letting you in she may need to set the sticky group bit too.</p>
