<p>Hi can you tell when it is coming this error.
I think you gave environment set wrong<br>
You should enter bellow command: </p>

<p>export HBASE_HOME="/usr/lib/hbase/hbase-0.94.18"</p>

<p>Then try hbase it will work.
If you want shell script you can download this lik :: <a href="https://github.com/tonyreddy/Apache-Hadoop1.2.1-SingleNode-installation-shellscript" rel="nofollow">https://github.com/tonyreddy/Apache-Hadoop1.2.1-SingleNode-installation-shellscript</a>

<td class="answercell">
    <div class="post-text" itemprop="text">
<p>sed "s/&lt;/configuration>/\nhbase.rootdir&lt;/name>\nhdfs://'$c':54310/hbase&lt;/value>\n&lt;/property>\n\nhbase.cluster.distributed&lt;/name>\ntrue&lt;/value>\n&lt;/property>\n\nhbase.zookeeper.property.clientPort&lt;/name>\n2181&lt;/value>\n&lt;/property>\n\nhbase.zookeeper.quorum&lt;/name>\n'$c'&lt;/value>\n&lt;/property>\n&lt;/configuration>/g" -i.bak hbase/conf/hbase-site.xml</p>

<p>sed 's/localhost/'$c'/g' hbase/conf/regionservers -i</p>

<p>sed 's/#\ export\ HBASE_MANAGES_ZK=true/export\ HBASE_MANAGES_ZK=true/g' hbase/conf/hbase-env.sh -i</p>

<p>Yes just type this tree commands and you need change replace $c to your hostname.
Then try it will work.</p>
