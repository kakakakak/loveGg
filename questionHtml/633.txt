<td class="answercell">
    <div class="post-text" itemprop="text">
<p>Your question is a bit cryptic, but from the title and question body combined I deduce you want to start X windows as root. Simply pasting your error into google led me to the following <a href="http://www.linuxquestions.org/questions/linux-software-2/xlib-connection-to-0-0-refused-by-server-331779/" rel="nofollow">link</a>. This suggests:</p>

<p><em>quote:</em>
As normal user, you can run:</p>

<pre><code>xhost local:root
</code></pre>

<p>and then root will be able to open X programs. You can edit <code>/root/.bashrc</code> and add the following line:</p>

<pre><code>export XAUTHORITY=/home/name/.Xauthority
</code></pre>

<p>Where "name" is the user account name that you use. After editing <code>/root/.basrc</code>, you won't have to issue the command <code>xhost local:root</code> anymore after X starts.</p>
