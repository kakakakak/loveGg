<td class="answercell">
    <div class="post-text" itemprop="text">
<p>I recently did a project that was heavily based on Avro data and not having used this data format before, I had to start from scratch. You are right in that it is rather hard to get much help from online sources when getting started with Avro. The material that I would recommend to you is:</p>

<ul>
<li>By far, the most helpful source that I found was the Avro section (p103-p116) in Tom White's <a href="http://oreilly.com/catalog/9780596521981">Hadoop: The Definitive Guide</a> book as well as his <a href="https://github.com/tomwhite/hadoop-book/tree/master/avro">Github page</a> for the code he uses in the book. </li>
<li>For additional code examples I looked at Ron Bodkin's Github page <a href="https://github.com/rbodkin/avro-mr-sample">avro-mr-sample</a>. </li>
<li>In my case I used Python for reading and writing Avro files and for that I used this <a href="http://www.harshj.com/2010/04/25/writing-and-reading-avro-data-files-using-python/">tutorial</a>.</li>
<li>Even though it is obvious, I will add the link to the <a href="http://mail-archives.apache.org/mod_mbox/avro-user/">Avro Users mailing list</a>. There is a ton of information to be found there and after I had read the above material and implemented a bunch of code, I found myself spending hours looking through the archives.</li>
</ul>

<p>Finally, my last suggestion to you is to use <strong>Avro 1.4.1 with Hadoop 0.20.2</strong> and ONLY that combination. I had some major issues getting my code to run using Hadoop 0.21 and more recent Avro versions.</p>
