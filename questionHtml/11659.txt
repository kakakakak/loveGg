    <div class="post-text" itemprop="text">

<p>Are there any packages in R for Facebook that are similar to twitteR?  I would like to analyze Facebook data just like the twitteR package does for Twitter.</p>

<p>My objective is to analyze how the facebookers talking about a certain keyword. Alike analysis had been done for Twitter with twitteR package. For example: <a href="http://www.cloudstat.org/index.php?do=/kafechew/blog/twitter-listening-apps/">Twitter Listening Robot</a>. This analysis can tell us who is tweeting, retweeting, who is the top users, when they are tweeting, who is tweeting whom, wordcloud and so.</p>

<p>I would like to replicate the same for Facebook. I just started researching about this and found that lack of public Facebook analysis using R.</p>

<p>Some useful resources for this topic I found: </p>

<ol>
<li><a href="https://gist.github.com/1634662">Facebook Mining</a></li>
<li><a href="http://applyr.blogspot.com/2012/01/mining-facebook-data-most-liked-status.html?spref=tw">Mining Facebook Data: Most "Liked" Status and Friendship Network </a></li>
</ol>

<p>Updated 23 June:</p>

<p>Since lack of information about the public facebook graph analysis, I would like to start the ball rolling, here are some source code or information might be useful. Join me if you're interested.</p>

<ul>
<li><a href="http://www.cloudstat.org/index.php?do=/kafechew/blog/facebook-graph-scrapping-social-info-with-r/">Facebook Graph: Scrapping Social Info with R</a> is the Facebook Graph (#API) parameter summary.</li>
<li><a href="http://www.cloudstat.org/index.php?do=/kafechew/blog/most-likes-facebook-post-for-keyword-umno-3/">Most LIKES facebook post for keyword UMNO 3</a> is a sample R script with Facebook Graph application.</li>
</ul>
