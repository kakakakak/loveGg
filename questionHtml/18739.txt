    <div class="post-text" itemprop="text">

<p>This might seem like an inane question but with all the buzz about big data I was curious as to how the typical datasets used in big data are sourced? Twitter keywords seem to be a common source - but what are the origins of the huge twitter feed files that get analysed? I saw an example where there was an analysis of election related words like Obama and Romney..has someone queried the Twitter API and downloaded effectively several terabytes of Tweets? Does Twitter even want people hitting their servers that hard? Or is this data already 'owned' by the companies doing the analytics. It might sound an odd scenario but most of the articles I have seen are fuzzy about these basic physical steps. Any links to good articles or tutorials that address these fundamental issues would be most appreciated</p>

<td class="answercell">
    <div class="post-text" itemprop="text">
<p>Here are some ideas to get sources of Big Data:</p>

<ul>
<li>As you pointed <a href="https://twitter.com/" rel="nofollow">Twitter</a> is a great place to grab data and there's a lot of useful analysis to do. If you're taking <a href="https://www.coursera.org/course/datasci" rel="nofollow">the online course about Data Science</a> one of the assignments is actually how to get live data from Twitter to analyze so I would recommend you take a look at <a href="https://class.coursera.org/datasci-001/assignment/view?assignment_id=3" rel="nofollow">this assignment</a> as the process of getting live Twitter data is very detailed. You could let the live stream run for days and it would probably generate Gigabytes worth of data the longer it runs.</li>
<li>If you have a website you could get web server logs. It might not be a lot if it's a small website, but for large websites who see a lot of traffic this is a huge source of data. Think about what you could do if you had StackOverflow web server logs...</li>
<li>Oceanographic data which you can find at <a href="http://marinexplore.org/" rel="nofollow">Marinexplore</a>, they have some huge datasets available that you can download and analyze yourself if you want to analyze ocean data.</li>
<li>Web crawling data, for example used by search engines. You can see some open data coming from web crawl at <a href="http://commoncrawl.org/" rel="nofollow">Common Crawl</a> which is already on Amazon S3 so ready to get your Hadoop jobs running on it ! You could also get data from Wikipedia <a href="http://en.wikipedia.org/wiki/Wikipedia%3aDatabase_download" rel="nofollow">here</a>.</li>
<li>Genomic data is now available on a very large scale and you can find genome data on <a href="http://www.1000genomes.org/" rel="nofollow">the 1000 genomes project</a> via FTP.</li>
<li>...</li>
</ul>

<p>More generally I would advise you look at <a href="http://aws.amazon.com/datasets" rel="nofollow">Amazon AWS datasets</a> which has a bunch of big datasets on various topics if you're not just looking at Twitter but Big Data in a more general context.</p>
