<td class="answercell">
    <div class="post-text" itemprop="text">
<p>Force option is not there for either of the commands (<code>get</code> /<code>copytolocal</code>).</p>

<p>Below are some options</p>

<pre><code>option 1 .
</code></pre>

<p>Remove the file on localmachine with <a href="http://en.wikipedia.org/wiki/Rm_%28Unix%29" rel="nofollow">rm</a> command.</p>

<p>and use <code>copyToLocal/get</code></p>

<pre><code>option 2
</code></pre>

<p>or rename your local file to  new name so that you can have the file with same name as on cluster.
use <a href="http://en.wikipedia.org/wiki/Mv" rel="nofollow">mv</a> command for that.</p>

<p>And use <code>get/copyTolocal</code> command</p>

<pre><code>option 3
</code></pre>

<p>rename the file there on the cluster itself and use copytolocal</p>

<pre><code>hadoop fs -mv [oldpath] [newpath]
hadoop fs -copytolocal [newpath] .
</code></pre>
