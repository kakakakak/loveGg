"/usr/local/hadoop/bin/hadoop: line 133: /usr/lib/jvm/jdk//bin/java: No such file or directory"

my .bashrc</p>

<pre><code>export JAVA_HOME=/usr/lib/jvm/jdk/

export PATH=$PATH:$HADOOP_INSTALL/bin

export HADOOP_MAPRED_HOME=$HADOOP_INSTALL
export HADOOP_COMMON_HOME=$HADOOP_INSTALL
export HADOOP_HDFS_HOME=$HADOOP_INSTALL
export YARN_HOME=$HADOOP_INSTALL
###end of paste
</code></pre>

<p>After that I opened hadoop-env.sh and pasted this ,the java home</p>

<pre><code>export JAVA_HOME=/usr/lib/jvm/jdk/
</code></pre>

<pre><code>"/usr/local/hadoop/bin/hadoop: line 133: /usr/lib/jvm/jdk//bin/java: No such file or directory"

<ol>
<li>First remove <code>/</code> from the end <code>/usr/lib/jvm/jdk/</code> in bot <code>bashrc</code> and <code>hadoop-env.sh</code></li>

see if it has java folder or not. If its not there then check if u have made the correct soft link.
You must create a soft link for folder that has java in it so check before this command:</p>

<p>$ cd /usr/lib/jvm
$ ln -s java-7-openjdk-amd64 jdk</p></li>
</ol>

<p>in above step as u might have seen in the tutorial change as following</p>

<pre><code>$ cd /usr/lib/jvm
$ ln -s java-7-openjdk-amd64/ jdk
</code></pre>

<p>the 7 here is dependent on verion of jdk u have so check that and change accordingly.
I have jdk 6 so i changed for java-6-<strong>**</strong></p>

<p>This is error due to $JAVA_HOME variable. Change this variable path. you will be free from the error. </p>

<p>go to .bashrc using this command </p>

<pre><code>vim ~/.bashrc 
</code></pre>

<p>Change the JAVA_HOME variable </p>

<pre><code> export JAVA_HOME=/usr/lib/jvm/jdk
 export PATH=PATH:$PATH/bin
</code></pre>

<p>if you have jdk 8 then replace jdk with java-8-oracle .</p>

<pre><code> export JAVA_HOME=/usr/lib/jvm/java-8-oracle
 export PATH=PATH:$PATH/bin 
</code></pre>

<p>restart you terminal and check the java command first, then hadoop command. </p>
