<p>I am trying to write a simple "Hello World" kind of application using spark streaming and RabbitMq, in which Apache Spark Streaming will read message from RabbitMq via the <a href="https://github.com/Stratio/rabbitmq-receiver" rel="nofollow">RabbitMqReceiver</a> and print it in the console. But some how I am not able to print the string read from Rabbit Mq into console. The spark streaming code is printing the message below:-</p>

<pre><code>Value Received BlockRDD[1] at ReceiverInputDStream at RabbitMQInputDStream.scala:33
Value Received BlockRDD[2] at ReceiverInputDStream at RabbitMQInputDStream.scala:33
</code></pre>

<p>The message is sent to the rabbitmq via the simple code below:-
</p>

<pre><code>package helloWorld;

import com.rabbitmq.client.Channel;
import com.rabbitmq.client.Connection;
import com.rabbitmq.client.ConnectionFactory;

public class Send {

  private final static String QUEUE_NAME = "hello1";

  public static void main(String[] argv) throws Exception {
    ConnectionFactory factory = new ConnectionFactory();
    factory.setHost("localhost");
    Connection connection = factory.newConnection();
    Channel channel = connection.createChannel();

    channel.queueDeclare(QUEUE_NAME, false, false, false, null);
    String message = "Hello World! is a code. Hi Hello World!";
    channel.basicPublish("", QUEUE_NAME, null, message.getBytes("UTF-8"));
    System.out.println(" [x] Sent '" + message + "'");

    channel.close();
    connection.close();
  }
}
</code></pre>

<p>I am trying to read messages via Apache Streaming as shown below:-</p>

<pre><code>package rabbitmq.example;

import java.util.*;

import org.apache.spark.SparkConf;
import org.apache.spark.api.java.JavaRDD;
import org.apache.spark.api.java.function.Function;
import org.apache.spark.streaming.Durations;
import org.apache.spark.streaming.api.java.JavaReceiverInputDStream;
import org.apache.spark.streaming.api.java.JavaStreamingContext;

import com.stratio.receiver.RabbitMQUtils;

public class RabbitMqEx {

    public static void main(String[] args) {
        System.out.println("Creating    Spark   Configuration");
        SparkConf conf = new SparkConf();
        conf.setAppName("RabbitMq Receiver Example");
        conf.setMaster("local[2]");

        System.out.println("Retreiving  Streaming   Context from    Spark   Conf");
        JavaStreamingContext streamCtx = new JavaStreamingContext(conf,
                Durations.seconds(2));      

        Map&lt;String, String&gt;rabbitMqConParams = new HashMap&lt;String, String&gt;();
        rabbitMqConParams.put("host", "localhost");
        rabbitMqConParams.put("queueName", "hello1");       
        System.out.println("Trying to connect to RabbitMq");
        JavaReceiverInputDStream&lt;String&gt; receiverStream = RabbitMQUtils.createJavaStream(streamCtx, rabbitMqConParams);
        receiverStream.foreachRDD(new Function&lt;JavaRDD&lt;String&gt;, Void&gt;() {           
            @Override
            public Void call(JavaRDD&lt;String&gt; arg0) throws Exception {               
                System.out.println("Value Received " + arg0.toString());
                return null;
            }
        } );        
        streamCtx.start();
        streamCtx.awaitTermination();
    }
}
</code></pre>

<p>The output console only has message like the following:-</p>

<pre><code>Creating    Spark   Configuration
Retreiving  Streaming   Context from    Spark   Conf
Trying to connect to RabbitMq
Value Received BlockRDD[1] at ReceiverInputDStream at RabbitMQInputDStream.scala:33
Value Received BlockRDD[2] at ReceiverInputDStream at RabbitMQInputDStream.scala:33
</code></pre>

    <div class="post-text" itemprop="text">
<p>This issue has been answered <a href="https://github.com/Stratio/rabbitmq-receiver/issues/20" rel="nofollow">here</a> by one of the authors of <code>rabbitmq-receiver</code>. Just quoting it below:-</p>

<blockquote>
  <p>Hi,</p>
  
  <p>You have declared the queue as non durable. The recevier expects to
  read from a durable queue. Try to declare it in this way:</p>
  
  <p><code>channel.queueDeclare(QUEUE_NAME, false, true, false, null);</code></p>
</blockquote>

<p>Also I found out declaring the queue as passive also works fine like below:-
<code>channel.queueDeclare(QUEUE_NAME, true, false, false, null)</code></p>
