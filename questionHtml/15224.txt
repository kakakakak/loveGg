<p>I am trying to download in Java a file from S3 to HDFS. This is my code:</p>

<pre><code>Configuration conf = new Configuration();
    // set the hadoop config files
    conf.addResource(new Path("/etc/hadoop/conf/core-site.xml"));
    conf.addResource(new Path("/etc/hadoop/conf/hdfs-site.xml"));

    Path path = new Path("hdfs://MY_IP_ADDRESS:8020/user/ubuntu/");
    FileSystem fs = path.getFileSystem(conf);

    try {
        Download download = myTransferManager.download(fs, "my_bucket", "test_cu_jmen2", new Path("test4.txt"));

        download.waitForCompletion();

        System.out.println("Download Progress\t" + download.getProgress().toString());
        System.out.println("Download State\t" + download.getState());
        System.out.println("Download Description\t" + download.getDescription());
        System.out.println("Download object metadata\t" + download.getObjectMetadata().toString());

    }
    catch(InterruptedException e)
    {
        e.printStackTrace();
    }
    catch(AmazonS3Exception e)
    {
        System.out.println(e.toString());
        System.out.println(e.getCause());
        System.out.println(e.getLocalizedMessage());
        System.out.println(e.getErrorCode());
        System.out.println(e.getErrorType());
        System.out.println(e.getMessage());
        e.printStackTrace();
    }
</code></pre>

<p>This is what I get in my console:</p>

<pre><code>Download Progress   com.amazonaws.services.s3.transfer.internal.TransferProgressImpl@3734b28b
Download State Canceled
Download Description    Downloading from my_bucket/test_cu_jmen2
Download object metadata    com.amazonaws.services.s3.model.ObjectMetadata@64b5dffd
</code></pre>

<p>Any ideas how to debug this? I want to find out why the download got cancelled.</p>

Serban</p>
