    <div class="post-text" itemprop="text">
<p>I think what you want to do is this: </p>

<pre><code>daily = load 'NYSE_daily.txt' using PigStorage(',') as (exchange:chararray, symbol:chararray, date:chararray, open:float, high:float , low:float, close:float, volume:float, adj_close:float);
qtr_filter = FILTER daily by date &gt;= '2009-10-01' and date &lt;= '2009-12-31';
grouped_by_stock = group qtr_filter by symbol;
</code></pre>

<p>each line of your output will be for 1 stock symbol and will contain all the daily data for that symbol.  It will not be ordered though so if you want to use that data in order must order it first.</p>
