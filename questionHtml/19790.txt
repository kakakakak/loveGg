<p>I have my java file under org/myorg</p>

<pre><code>  package org.myorg;
  import java.io.*;
  import java.util.*;
  import java.net.*;
  import org.apache.hadoop.fs.*;
  import org.apache.hadoop.conf.*;
  import org.apache.hadoop.io.*;
  import org.apache.hadoop.mapred.*;
  import org.apache.hadoop.util.*;

  public class Write{
    public static void main (String [] args) throws Exception{
            try{
                    Path pt=new Path("hdfs:localhost:8020/user/jfor/out/abc");
                    FileSystem fs = FileSystem.get(new Configuration());
                    BufferedWriter br=new BufferedWriter(new OutputStreamWriter(fs.create(pt,true)));
                                               // TO append data to a file, use fs.append(Path f)
                    String line;
                    line="Disha Dishu Daasha";
                    System.out.println(line);
                    br.write(line);
                    br.close();
            }catch(Exception e){
                    System.out.println("File not found");
            }
    }
}
</code></pre>

<p>I compiled it using</p>

<pre><code> javac -classpath hadoop-0.20.1-dev-core.jar org/myorg/Write.java
</code></pre>

<p>from the parent directory of <code>org</code>.</p>

<p>Write.class is created under <code>org/myorg</code></p>

<p>I run it using </p>

<p>java org.myorg.Write</p>

<p>from the parent directory of <code>org</code></p>

<p>It says</p>

<pre><code> Exception in thread "main" java.lang.NoClassDefFoundError: org/apache/hadoop/fs/FSDataOutputStream
    at java.lang.Class.getDeclaredMethods0(Native Method)
</code></pre>

<p>But i downloaded jar from <a href="http://www.java2s.com/Code/Jar/h/Downloadhadoop0201corejar.htm" rel="nofollow">this link</a>. This link says FSDataOutputStream class is present.</p>

<p>What could be the reason?</p>

<td class="answercell">
    <div class="post-text" itemprop="text">
<p>You don't have the classpath set for execution - you have it set during compile:</p>

<p>Windows:</p>

<pre><code>java -cp .;hadoop-0.20.1-dev-core.jar org.myorg.Write
</code></pre>

<p>UNIX:</p>

<pre><code>java -cp .:hadoop-0.20.1-dev-core.jar org.myorg.Write
</code></pre>

<p>You have to specify the classpath (-cp) to include the current directory, in your case, as well as the hadoop jar file.</p>

    <div class="post-text" itemprop="text">
<p>You have to give the exact path where the <code>jar</code> is present </p>

<pre><code>java -cp .:give_the_path_to_the_jar org.myorg.Write
</code></pre>

<p>for Example if it is present in my desktop</p>

<pre><code>java -cp .:/home/neeraj/Desktop/hadoop-0.20.1-dev-core.jar org.myorg.Write
</code></pre>

<blockquote>
  <p><code>.</code>  represent the current directory</p>
</blockquote>

<p><strong>Note :</strong> These above example is for linux(ubuntu) , if you need to use in Windows than you have to use <code>;</code> instead of <code>:</code></p>
