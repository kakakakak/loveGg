<p>I am using eclipse to write mapreduce program. I imported hadoop library 
(hadoop-0.13.0-core.jar)</p>

<p>I imported Mapper class import org.apache.hadoop.mapred.Mapper;
there is no error in this but when I wrote this program
source of this is <a href="http://developer.yahoo.com/hadoop/tutorial/module3.html" rel="nofollow">http://developer.yahoo.com/hadoop/tutorial/module3.html</a></p>

<pre><code>public class WordCountMapper extends MapReduceBase
    implements Mapper&lt;LongWritable, Text, Text, IntWritable&gt; {

  private final IntWritable one = new IntWritable(1);
  private Text word = new Text();

  public void map(WritableComparable key, Writable value,
      OutputCollector output, Reporter reporter) throws IOException {

    String line = value.toString();
    StringTokenizer itr = new StringTokenizer(line.toLowerCase());
    while(itr.hasMoreTokens()) {
      word.set(itr.nextToken());
      output.collect(word, one);
    }
  }
}
</code></pre>

<p>it gives me error The type Mapper is not generic; it cannot be 
 parameterized with arguments </p>
