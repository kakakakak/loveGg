<p>How can I find which Hive version I am using from the command prompt. Below is the details-</p>

<p>I am using Putty to connect to hive table and access records in the tables. So what I did is- I opened Putty and in the host name I typed- <code>leo-ingesting.vip.name.com</code> and then I click <code>Open</code>. And then I entered my username and password and then few commands to get to Hive sql. Below is the list what I did</p>

<pre><code>$ bash
bash-3.00$ hive
Hive history file=/tmp/rkost/hive_job_log_rkost_201207010451_1212680168.txt
hive&gt; set mapred.job.queue.name=hdmi-technology;
hive&gt; select * from table LIMIT 1;
</code></pre>

<p>So is there any way from the command prompt I can find which hive version I am using and Hadoop version too?</p>

<p>You can not get hive version from command line.</p>

<p>You can checkout hadoop version as mentioned by Dave.</p>

<p>Also if you are using cloudera distribution, then look directly at the libs:</p>

<p>ls /usr/lib/hive/lib/ and check for hive library</p>

<pre><code>hive-hwi-0.7.1-cdh3u3.jar
</code></pre>

<p>You can also check the compatible versions here:</p>

<p><a href="http://www.cloudera.com/content/cloudera/en/documentation/cdh5/v5-1-x/CDH-Version-and-Packaging-Information/CDH-Version-and-Packaging-Information.html" rel="nofollow">http://www.cloudera.com/content/cloudera/en/documentation/cdh5/v5-1-x/CDH-Version-and-Packaging-Information/CDH-Version-and-Packaging-Information.html</a></p>
