<p>Ok, After playing around for a while I finally got it and hopefully this will work for you aswell. That tutorial builds spark, where they do provide prebuilt binaries. I'm using Spark 1.2.0 just as a note (1.4.1 wouldn't work for me)</p>

<p>This is on Ubuntu 15.04 but should work on 14.04 the same</p>

<p>1) Remove the following lines from your bashrc</p>

<pre><code>export SCALA_HOME=/usr/local/src/scala/scala-2.10.4
export PATH=$SCALA_HOME/bin:$PATH
</code></pre>

<p>2) Remove and reinstall scala</p>

<pre><code>sudo rm -rf /usr/local/src/scala
# The following line is only needed if you installed scala another way, if so remove the #
# sudo apt-get remove scala-library scala
wget http://www.scala-lang.org/files/archive/scala-2.11.7.deb
sudo dpkg -i scala-2.11.7.deb
sudo apt-get update
sudo apt-get install scala
</code></pre>

<p>3) Download <strong>PreBuilt</strong> Spark and extract</p>

<pre><code>wget http://d3kbcqa49mib13.cloudfront.net/spark-1.2.0-bin-hadoop2.4.tgz
tar -xzvf spark-1.2.0-bin-hadoop2.4.tgz 
</code></pre>

<p>4) Run spark-shell</p>

<pre><code>cd spark-1.2.0-bin-hadoop2.4/
./bin/spark-shell
</code></pre>

<p>In the <strong>SPARK_HOME/sbt/sbt-launch-lib.bash</strong> script replace line 53 to line 57 with following</p>

<pre><code>if hash curl 2&gt;/dev/null; then
  (curl --fail --location --silent ${URL1} &gt; ${JAR_DL} ||\
   (rm -f "${JAR_DL}" &amp;&amp; curl --fail --location --silent ${URL2} &gt; ${JAR_DL})) &amp;&amp; \
   mv "${JAR_DL}" "${JAR}"
elif hash wget 2&gt;/dev/null; then
  (wget --quiet ${URL1} -O ${JAR_DL} ||\
   (rm -f "${JAR_DL}" &amp;&amp; wget --quiet ${URL2} -O ${JAR_DL})) &amp;&amp;\
   mv "${JAR_DL}" "${JAR}"
else
</code></pre>

<pre><code>sbt/sbt assembly
