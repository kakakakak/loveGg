<td class="answercell">
    <div class="post-text" itemprop="text">
<p>You most probably have speculative execution on, and two different attempts for reduce task 1 are trying to write to the path <code>/home/users/mlakshm/alop176/data-r-00001</code>. This probably succeeds for smaller tasks as they finish before hadoop speculatively executes a second attempt.</p>

<p>I see your implementation of MultipleOutputs is custom (<code>com.a.MultipleOutputs</code>), you should be writing all HDFS data to the tasks working directory and letting the OutputComitter move it to the final output directory upon output commit. If you're able to, pastebin the code and we can take a look.</p>
