    <div class="post-text" itemprop="text">

<p>I want to do something similar as <a href="http://stackoverflow.com/questions/13728289/r-scrape-a-blog-for-title-number-of-comments-and-likes">this SO question</a> using <em>R or any other similar software</em>.</p>

<p>The additional requirements I want is:</p>

<ol>
<li>scrape for <strong>keywords in the blogpost content</strong></li>
<li>and count their occurrences</li>
</ol>

<p>For simplicity sake, let us assume the blog in question is also www.jamesaltucher.com</p>

<p>By keywords, I mean any words that are the usual words google ignores like, "the", "an", "a", "of", etc.</p>

<p>Please assume that the blogs are public and I do not have admin account.</p>

<p>By content, I mean the content inside the blog post itself. Not necessarily comments.</p>

<p><strong>UPDATE:</strong>
Currently, I can extract all the blogpost urls from www.jamesaltucher.com/sitemap.xml</p>

<pre><code>library(XML)
library(RCurl)
url.link &lt;- 'http://www.jamesaltucher.com/sitemap.xml'
blog &lt;- getURL(url.link)
blog          &lt;- htmlParse(blog, encoding = "UTF-8")
titles  &lt;- xpathSApply (blog ,"//loc",xmlValue)             ## titles
</code></pre>

<p>This is the screenshot.</p>

<p><img src="http://i.stack.imgur.com/uCkSt.png" alt="enter image description here"></p>

<p><strong>My next step is to iterate through all the entries in <code>titles</code> and extract the content for keyword frequency counting.</strong></p>

<p>How do I go about doing that?</p>

<p>You can do this easily in Python and maybe you can combine Python script below in R using rpy or </p>

<pre><code>scan(pipe("python scraper.py | cat"))
</code></pre>

<p>The most difficult part must be correctly get the content of the blog... If you simply do soup.text. you might get some source code of the blog instead of the real content. You need to correctly locate your content using HTML tags, id, class.. etc. </p>

<p>For more information, please refer to this page of <a href="http://www.crummy.com/software/BeautifulSoup/bs4/doc/#searching-the-tree" rel="nofollow">BeautifulSoup Search the tree</a>. </p>

<pre><code>import urllib2
from collections import Counter
from bs4 import BeautifulSoup

# get the html page of your target blog
soup = BeautifulSoup(urllib2.urlopen('http://datafireball.com/'))
# first, you need to retrieve the content of the blog
content = soup.find("div", {"id":"content"}).text.encode('utf-8')
# second, you count the frequency
cnt = Counter(content.split())
# print out the result to verify it works
print cnt
</code></pre>

<p>In R, something like this will work for you:</p>

<pre><code>out &lt;- lapply(titles, function(url) {
  content &lt;- getURL(url)
  content &lt;- gsub("&lt;.*?&gt;", "",content)
  content &lt;- gsub("[[:space:][:punct:][:digit:]]+", " ",content)
  content &lt;- tolower(content)
  table(strsplit(content,' '))
})
</code></pre>

<p>The result is a list of tables, though you could have it just return the output of <code>strsplit</code> directly instead.</p>

<p>You'll probably have to go through and remove words you don't like (e.g., <code>if</code>, <code>the</code>, etc.) but to do that you'll need a complete dictionary of such words to remove.</p>
