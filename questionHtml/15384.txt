<td class="answercell">
    <div class="post-text" itemprop="text">
<p>If you are using spark with hdp, then we have to do following things.</p>

<ol>
<li><p>Add these entries in your <strong>$SPARK_HOME/conf/spark-defaults.conf</strong></p>

<p>spark.driver.extraJavaOptions  -Dhdp.version=2.2.0.0-2041 <strong>(your installed HDP version)</strong></p>

<p>spark.yarn.am.extraJavaOptions -Dhdp.version=2.2.0.0-2041 <strong>(your installed HDP version)</strong></p></li>
<li><p>create <strong>java-opts</strong> file in <strong>$SPARK_HOME/conf</strong> and add the installed HDP version in that file like</p></li>
</ol>

<p>-Dhdp.version=2.2.0.0-2041 <strong>(your installed HDP version)</strong></p>

<p>to know hdp verion please run command <strong>hdp-select status hadoop-client</strong> in the cluster</p>
