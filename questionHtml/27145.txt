    <div class="post-text" itemprop="text">
<p>I used Hadoop-2.7, Hive-1.1, HBase-1.0.0
Use the following syntax:</p>

<p><code>hbase shell</code></p>

<p><code>create 'people', 'info'</code></p>

<pre>
put 'people', '100', 'info:name', 'naga'
put 'people', '101', 'info:name', 'siva'
put 'people', '100', 'info:age',  30
put 'people', '101', 'info:age',  35
</pre>

<p>hive</p>

<p><strong><code>CREATE external TABLE people(rid string, name string, age int) 
STORED BY 'org.apache.hadoop.hive.hbase.HBaseStorageHandler' WITH SERDEPROPERTIES ("hbase.columns.mapping" = ":key,info:name,info:age", "hbase.table.name" = "people");</code></strong></p>

<pre>
hive> select * from people;
OK
100 naga    30
101 siva    35
Time taken: 0.115 seconds, Fetched: 2 row(s)
</pre>

<p><strong>Note:</strong> If it is not working, then check the jars compatibility and any missing jars.</p>

<p><strong>Update:</strong> Even this one also works fine:</p>

<p><strong><code>CREATE TABLE hbase_people(rid string, name string, age int) 
 STORED BY 'org.apache.hadoop.hive.hbase.HBaseStorageHandler' WITH  SERDEPROPERTIES ("hbase.columns.mapping" = ":key,info:name,info:age", "hbase.table.name" = "hbase_people");</code></strong></p>

<p>OK</p>

<p>Time taken: 0.48 seconds</p>
