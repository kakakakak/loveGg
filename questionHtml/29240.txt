    <div class="post-text" itemprop="text">
<p><a href="http://hadoop.apache.org/">Hadoop</a> refers to an ecosystem of software packages, including MapReduce, HDFS, and a whole host of other software packages to support the import and export of data into and from HDFS (the Hadoop Distributed FileSystem). When someone says, "I have a Hadoop cluster," they generally mean a cluster of machines all running in this general ecosystem with a large distributed filesystem to support large scale computation.</p>

<p>NoSQL is referring to non-relational or at least non-SQL database solutions such as <a href="http://hbase.apache.org/">HBase</a> (also a part of the Hadoop ecosystem), <a href="http://cassandra.apache.org/">Cassandra</a>, <a href="http://mongodb.org/">MongoDB</a>, <a href="http://basho.com/products/riak-overview/">Riak</a>, <a href="http://couchdb.apache.org/">CouchDB</a>, and many others.</p>

<p>Hadoop - computing framework</p>

<p>NoSQL  - Not Only - SQL databases</p>

<p>According to Wikipedia:</p>

<p><a href="http://en.wikipedia.org/wiki/Hadoop">Hadoop</a>:</p>

<blockquote>
  <p>Apache Hadoop is an open-source software framework that supports data-intensive distributed applications, licensed under the Apache v2 license.<a href="http://www.oracle.com/technetwork/articles/bigdata/nosql-hadoop-1654158.html">1</a> It enables applications to work with thousands of computational independent computers and petabytes of data.</p>
</blockquote>

<p><a href="http://en.wikipedia.org/wiki/NoSQL">NoSQL</a>:</p>

<blockquote>
  <p>In computing, NoSQL (mostly interpreted as "not only SQL"<a href="http://www.oracle.com/technetwork/articles/bigdata/nosql-hadoop-1654158.html">1</a>) is a broad class of database management systems identified by its non-adherence to the widely used relational database management system model; that is, NoSQL databases are not primarily built on tables, and as a result, generally do not use SQL for data manipulation.</p>
</blockquote>

<p>They are not the same thing, but are related to data inten. Hadoop is an entire framework (that can be used with NoSQL DBMS <a href="http://www.oracle.com/technetwork/articles/bigdata/nosql-hadoop-1654158.html">like Oracle NoSQL</a>).</p>

<p>I think your questions leads in the wrong direction.</p>

<p>hadoop is the name for a distributed filesystem called hdfs and a possibility to use the map reduce paradigm for large scale calculations. Since yarn it is also possible to use a hadoop cluster with a lot more tasks (like storm, hive, etc.).</p>

<p>nosql refers to databases not queried by sql and not necessarily use tables etc.</p>

<p>you may want to have a look at the wiki pages of <a href="http://en.wikipedia.org/wiki/Hadoop" rel="nofollow">hadoop</a> and <a href="http://en.wikipedia.org/wiki/NoSQL" rel="nofollow">noSQL</a> for further understanding of the differences between both</p>
