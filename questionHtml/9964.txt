    <div class="post-text" itemprop="text">
<p>Permissions are not so easy in Hadoop usually. Couple of debugging points:</p>

<ul>
<li>Please check from which user you run your job and which user is 'visible' on Hadoop cluster.</li>
<li>Please check what happens inside method and what files are being modified.</li>
<li>Assure required permissions are met. If not, you have option either to disable HDFS permissions or 'represent' yourself as different user to Hadoop cluster.</li>
</ul>

<p>These questions could be useful for you:</p>

<ul>
<li><a href="http://stackoverflow.com/questions/11593374/permission-denied-at-hdfs">Permission denied at hdfs</a></li>
<li><a href="http://stackoverflow.com/questions/15941108/hdfs-access-from-remote-host-through-java-api-user-authentication">HDFS access from remote host through Java API, user authentication</a></li>
</ul>
