    <div class="post-text" itemprop="text">
<blockquote>
  <p>Can I directly process the data as these are tar files.</p>
</blockquote>

<p>Yes, see the below solution.</p>

<blockquote>
  <p>if yes, how to remove the data in the first four lines</p>
</blockquote>

<p>Starting <strong>Hive v0.13.0</strong>, There is a table property, <code>tblproperties ("skip.header.line.count"="1")</code> while creating a table to tell <code>Hive</code> the number of rows to ignore. To ignore first four lines - <code>tblproperties ("skip.header.line.count"="4")</code></p>

<pre><code>CREATE TABLE raw (line STRING)
   ROW FORMAT DELIMITED FIELDS TERMINATED BY '\t' LINES TERMINATED BY '\n';

CREATE TABLE raw_sequence (line STRING)
   STORED AS SEQUENCEFILE
   tblproperties("skip.header.line.count"="4");

LOAD DATA LOCAL INPATH '/tmp/test.tar' INTO TABLE raw;

SET hive.exec.compress.output=true;
SET io.seqfile.compression.type=BLOCK; -- NONE/RECORD/BLOCK (see below)

INSERT OVERWRITE TABLE raw_sequence SELECT * FROM raw;
</code></pre>

<p>To view the data: </p>

<pre><code>select * from raw_sequence
</code></pre>

<p>Reference: <a href="https://cwiki.apache.org/confluence/display/Hive/CompressedStorage" rel="nofollow">Compressed Data Storage</a></p>

<td class="answercell">
    <div class="post-text" itemprop="text">
<p>Follow the below steps to achieve your goal:</p>

<ol>
<li>Copy the data(ie.tar file) to the client system where hadoop is installed.</li>
<li>Untar the file and manually remove the description and save it in local.</li>
<li><p>Create the metadata(i.e table) in hive based on the description.<br>
<strong>Eg</strong>: If the description contains emp_id,emp_no,etc.,then create table in hive using this information and also make note of field separator used in the data file and use the corresponding  field separator in create table query. Assumed that file contains two columns which is separated by comma then below is the syntax to create the table in hive.</p>

<pre><code>  Create table tablename (emp_id int, emp_no int)
  Row Format Delimited
  Fields Terminated by ','
</code></pre></li>
<li><p>Since, data is in structured format, you can load the data into hive table using the below command.
<code>LOAD DATA LOCAL INPATH '/LOCALFILEPATH' INTO TABLE TABLENAME</code>.</p></li>
<li><p>Now, local data will be moved to hdfs and loaded into hive table.</p></li>
<li><p>Finally, you can query the hive table using <code>SELECT * FROM TABLENAME;</code></p></li>
</ol>
