    <div class="post-text" itemprop="text">
<p>ok, you need a few more props fro hdfs sink:</p>

<pre><code>a1.sinks.k2.hdfs.filePrefix = [your prefix]
a1.sinks.k2.hdfs.fileSuffix = .[your suffix]
</code></pre>

<p>suffix would be .tsv or .csv for instance, while prefix can be anything - you can also use variables for date and time - this requires that you use the timestamp interceptor. You can also create your own interceptor and generate your own variables into your file name. If you omit this, flume will add its own sequence between prefix and suffix.</p>

<p>As an addition to our previous comments, the props to disable rollovers are the following:</p>

<pre><code>a1.sinks.k2.rollInterval = 0
a1.sinks.k2.rollSize = 0
a1.sinks.k2.rollCount = 0
a1.sinks.k2.idleTimeout = 0
</code></pre>

<p>to access the file name of the original file from your source, append the following in your hdfs sink config:</p>

<pre><code>a1.sinks.k2.hdfs.filePrefix = %{file}
</code></pre>

<p>to simplify your channel config, do the following:</p>

<pre><code>a1.channels.c1.type = memory
a1.channels.c1.capacity = 1000
</code></pre>
