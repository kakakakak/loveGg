<p>, I have installed<br>
1) scala-2.10.3</p>

<p>2) spark-1.0.0
    Changed spark-env.sh with below variables</p>

<pre><code>export SCALA_HOME=$HOME/scala-2.10.3

export SPARK_WORKER_MEMORY=16g
</code></pre>

<p>I can see Spark master.</p>

<p>3) shark-0.9.1-bin-hadoop1</p>

<pre><code>Changed shark-env.sh with below variables

export SHARK_MASTER_MEM=1g

SPARK_JAVA_OPTS=" -Dspark.local.dir=/tmp "

SPARK_JAVA_OPTS+="-Dspark.kryoserializer.buffer.mb=10 "

SPARK_JAVA_OPTS+="-verbose:gc -XX:-PrintGCDetails -XX:+PrintGCTimeStamps "

    export SPARK_JAVA_OPTS

    export HIVE_HOME=/usr/share/dse/hive

export HIVE_CONF_DIR="/etc/dse/hive"

export SPARK_HOME=/home/ubuntu/spark-1.0.0

export SPARK_MEM=16g

source $SPARK_HOME/conf/spark-env.sh
</code></pre>

<p>4) In DSE, Hive version is Hive 0.11
    Existing Hive-site.xml is </p>

<pre><code>    &lt;configuration&gt;

     &lt;!-- Hive Execution Parameters --&gt;
         &lt;property&gt;
                &lt;name&gt;hive.exec.mode.local.auto&lt;/name&gt;
                &lt;value&gt;false&lt;/value&gt;
             &lt;description&gt;Let hive determine whether to run in local mode automatically&lt;/description&gt;
        &lt;/property&gt;

        &lt;property&gt;
                &lt;name&gt;hive.metastore.warehouse.dir&lt;/name&gt;
              &lt;value&gt;cfs:///user/hive/warehouse&lt;/value&gt;
             &lt;description&gt;location of default database for the warehouse&lt;/description&gt;
        &lt;/property&gt;

        &lt;property&gt;
             &lt;name&gt;hive.hwi.war.file&lt;/name&gt;
              &lt;value&gt;lib/hive-hwi.war&lt;/value&gt;
                &lt;description&gt;This sets the path to the HWI war file, relative to ${HIVE_HOME}&lt;/description&gt;
          &lt;/property&gt;

          &lt;property&gt;
                &lt;name&gt;hive.metastore.rawstore.impl&lt;/name&gt;
                &lt;value&gt;com.datastax.bdp.hadoop.hive.metastore.CassandraHiveMetaStore&lt;/value&gt;
                &lt;description&gt;Use the Apache Cassandra Hive RawStore implementation&lt;/description&gt;
          &lt;/property&gt;

         &lt;property&gt;
                &lt;name&gt;hadoop.bin.path&lt;/name&gt;
                &lt;value&gt;${dse.bin}/dse hadoop&lt;/value&gt;
          &lt;/property&gt;

          &lt;!-- Set this to true to enable auto-creation of Cassandra keyspaces as Hive Databases --&gt;
          &lt;property&gt;
                &lt;name&gt;cassandra.autoCreateHiveSchema&lt;/name&gt;
                &lt;value&gt;true&lt;/value&gt;
          &lt;/property&gt;
    &lt;/configuration&gt;
</code></pre>

<p>5) while running Shark shell getting error: </p>

<pre><code> Unable to instantiate Org.apache.hadoop.hive.metastore.HiveMetaStoreClient
</code></pre>

<p>And</p>

<p>6) While running shark shell with -skipRddReload - I'm able to get Shark shell but not able to connect hive and not able execute any commands.</p>

<pre><code> shark&gt; DESCRIVE mykeyspace;
</code></pre>

<p>and getting error message: </p>

<pre><code>FAILED: Error in metastore: java.lang.RuntimeException: Unable to instantiate org.apache.haddop.hive.metastore.HiveMataStoreClient.
FAILED: Execution Error, return code 1 from org.apache.hadoop.hive.q1.exec.DDLTask.
</code></pre>
