<p>As your <code>hbase-site.xml</code> says - you have tried running hbase on hdfs also and now you are trying to run on local file system. <br>
Solution : run <code>hadoop.x.x.x/bin/start-dfs.sh</code> first, and then run <code>hbase.x.x.x/bin/start-hbase.sh</code> . It will now run as expected on local file system.<br></p>
