<p>Today I started working on rhdfs and rmr2 packages.  </p>

<p>mapreduce() function on a 1D vector worked well as expected.
<strong>piece of code on 1D vector</strong></p>

<pre><code>a1 &lt;- to.dfs(1:20)
a2 &lt;- mapreduce(input=a1, map=function(k,v) keyval(v, v^2))
a3 &lt;- as.data.frame(from.dfs(a2())
</code></pre>

<p>It returns following dataframe</p>

<pre><code>    Key  Val
1     1    1
2    10  100
3    11  121
4    12  144

</code></pre>

<p>Till now, it was fine.</p>

<p>But, While working on mapreduce function on mtcars dataset, I got the following error message. Unable to debug it further. Kindly give some clue to move ahead.</p>

<p><strong>My piece of code :</strong></p>

<pre><code>rs1 &lt;- mapreduce(input=mtcars, 
                  map=function(k, v) {
                      if (mtcars$hp &gt; 150) keyval("Bigger", 1) },
                  reduce=function(k, v)  keyval(k, sum(v))
                  )
</code></pre>

    <div class="post-text" itemprop="text">
<p>The data which you are passing for Keyval,thinks it as vector, its not single entity. Try to interpret from below code.</p>

<p><strong>Trying locally</strong></p>

<ul>
<li><p>loading data</p>

<pre><code>data(mtcars)
</code></pre></li>
<li><p>view few data lines</p>

<pre><code>head(mtcars)
hpTest=mtcars$hp # taking required data
print(hpTest)
</code></pre></li>
<li><p>Final sum</p>

<pre><code>sum(hpTest[which(hpTest&gt;150)])  # 2804
</code></pre></li>
</ul>

<p><strong>Running on Hadoop-MapReduce</strong></p>

<ul>
<li><p>exporting env variables</p>

<pre><code># requied
Sys.setenv(HADOOP_HOME="/home/trendwise/apache/hadoop-1.0.4");
Sys.setenv(HADOOP_CMD="/home/trendwise/apache/hadoop-1.0.4/bin/hadoop");

#optional
Sys.setenv(HADOOP_BIN="/home/trendwise/apache/hadoop-1.0.4/bin");
Sys.setenv(HADOOP_CONF_DIR="/home/trendwise/apache/hadoop-1.0.4/conf");
Sys.setenv(HADOOP_STREAMING='/home/trendwise/apache/hadoop-1.0.4/contrib/streaming/hadoop-streaming-1.0.4.jar')
Sys.setenv(LD_LIBRARY_PATH="/lib:/lib/x86_64-linux-gnu:/lib64:/usr/lib:/usr/lib64:/usr/local/lib:/usr/local/lib64:/usr/lib/jvm/jdk1.7.0_10/lib:/usr/lib/jvm/jdk1.7.0_10/jre/lib:/usr/lib/jvm/jdk1.7.0_10/jre/lib/amd64:/usr/lib/jvm/jdk1.7.0_10/jre/lib/amd64/server");
</code></pre></li>
<li><p>Loading library</p>

<pre><code>library(rmr2)
library(rhdfs)
</code></pre></li>
<li><p>initializing</p>

<pre><code>hdfs.init()
</code></pre></li>
<li><p>putting input into HDFS</p>

<pre><code>hpInput = to.dfs(mtcars$hp)
</code></pre></li>
<li><p>running MapReduce</p>

<pre><code>mapReduceResult &lt;- mapreduce(input=hpInput, 
          map=function(k, v) { keyval( rep(1,length(which(inputData &gt; 150))) ,v[which(v&gt;150)] )} ,
          reduce=function(k2, v2){  keyval(k2, sum(v2))}
</code></pre></li>
<li><p>viewing MR output</p>

<pre><code>from.dfs(mapReduceResult)
</code></pre></li>
<li><p>output</p>

<pre><code>$key
[1] 1

$val
[1] 2804    
</code></pre></li>
</ul>
