                <span class="comment-copy">Here&#39;s how you figure out the answer: The value section of <code>?searchTwitter</code> tells you it returns a <code>status</code> object and provides a link to <code>?status-class</code> which tells you what is returned. Alternatively, you could look at <code>str(searchTwitter(&quot;#rstats&quot;, n=10))</code> to see what is returned.</span>

<p>One alterantive to parse the result( as the answer above) , is to use argument <strong>since</strong> and <strong>until</strong>.</p>

<p>For example you can do : </p>

<pre><code> res &lt;- searchTwitter("#rstats",n=1000,since='2013-01-24',
                                       until='2013-01-28')
</code></pre>

<p>The <code>searchTwitter</code> is a wrapper to the JSON API of twitter. Take a look <a href="https://dev.twitter.com/docs/api/1/get/search" rel="nofollow">here</a> for more details of the argument and example of the JSON results.</p>
