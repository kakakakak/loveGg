<p>From <a href="https://groups.google.com/forum/#!topic/cascading-user/jkARDNH770E" rel="nofollow">google groups</a></p>

<pre><code>We don't compile for 2.11.0 since its super hard/impossible to have code compatible from 2.9 -&gt; 2.11.
We (at twitter) are still on 2.9.x for the moment so for now we can only support up to the 2.10.x scala's. Hopefully this will change over the next few months. 
</code></pre>

<p>Also the current scalding version(0.12) is compatible only with scala 2.10(<a href="http://mvnrepository.com/artifact/com.twitter/scalding-core_2.10" rel="nofollow">maven</a>)</p>

<p>As @Travis mentioned in the comment, Scalding for Scala version 2.11 is available in the scalding's develop branch(<a href="https://github.com/twitter/scalding" rel="nofollow">github</a>)</p>
