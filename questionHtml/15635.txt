<td class="answercell">
    <div class="post-text" itemprop="text">
<p>Try explicitly specify <code>sc.textFile("file:///path to the file/")</code>. The error occurs when Hadoop environment is set. </p>

<p>SparkContext.textFile internally calls <code>org.apache.hadoop.mapred.FileInputFormat.getSplits</code>, which in turn uses <code>org.apache.hadoop.fs.getDefaultUri</code> if schema is absent. This method reads "fs.defaultFS" parameter of Hadoop conf. If you set HADOOP_CONF_DIR environment variable, the parameter is usually set as "hdfs://..."; otherwise "file://".</p>
