    <div class="post-text" itemprop="text">
<p>I had same issue for Oracle DB connectivity.</p>

<p>I did following 4 things, but I think first 2 fixed it:</p>

<ol>
<li>When running your app, add driver jar to classpath:</li>
</ol>

<p><code>%YOUR_SPARK_HOME%/bin/spark-submit --jars file://c:/jdbcDrivers/ojdbc7.jar</code></p>

<ol start="2">
<li><p>Add driver in connection properties (options object in your case):</p>

<pre><code>    dbProps.put("driver", "oracle.jdbc.driver.OracleDriver");
</code></pre></li>
<li><p>Under %YOUR_SPARK_HOME%/conf/spark-defaults.conf file add:</p>

<p>spark.driver.extraClassPath = file://C:/jdbcDrivers/ojdbc7.jar</p></li>
<li><p>Under same conf also add:</p>

<p>spark.executor.extraClassPath = file://C:/jdbcDrivers/ojdbc7.jar</p></li>
</ol>

<p>I've got the connection to DB but working on some issues to insert into Oracle as of now.</p>
