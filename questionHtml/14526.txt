    <div class="post-text" itemprop="text">

<p>Is there someplace that lists which version of a Spring project works with a given version of the Spring Core framework?  </p>

<p>For example if I am using Spring Core 3.2.7, what version of Spring security can I use? </p>

<ul>
<li>If I use a lot of these components, then I would need to look each up in the maven pom or gradle build. It would be nice to be able look it up quickly. </li>
<li>Even so, looking it up only tells me that Spring Security 3.2.0 was built with Spring Core 3.2.6.  </li>
<li>Does that mean it will work with Spring Core 3.2.7 or do I need to downgrade my version of Core?</li>
</ul>

<p>Is there a Matrix that lists versions of the Spring Projects per the Spring Core version?</p>

    <div class="post-text" itemprop="text">
<p>After looking everywhere and not finding what I wanted, I compiled this matrix from the source code.  </p>
