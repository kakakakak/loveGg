<td class="answercell">
    <div class="post-text" itemprop="text">
<p>I got similar exception but "hdfs". I found the issue to be in application jar that I created using maven with dependencies did not have "services" specs for hdfs that comes with hadoop hdfs jar. I solved it using maven shade plugin (maven-shade-plugin) using its built in transformer "org.apache.maven.plugins.shade.resource.ServicesResourceTransformer". </p>

<p>I think maven without shade plugin would not put hdfs services specs for "org.apache.hadoop.fs.FileSystem" which is required by hdfs FileSystem. My guess is "maprfs" is another filesystem that needs to be specified in the METAINF/services folder. Try providing it manually. You can get it from hadoop hdfs jar.</p>

<p>Hope this helps.</p>
