<td class="answercell">
    <div class="post-text" itemprop="text">
<p>You need to run hadoop namenode -format as the hdfs-superuser. Probably the "hdfs" user itself.</p>

<p>The hint can be seen here:</p>

<pre><code>UserGroupInformation:  PriviledgedActionException as:root cause:java
</code></pre>

<p>Another  thing to consider: You really want to move your hdfs root to something other than /tmp. You will risk losing your hdfs contents when /tmp is cleaned (which could happen any time)</p>

<p><strong>UPDATE</strong> based on OP comments.</p>

<p>RE: JobTracker unable to contact NameNode: Please do not skip steps.  </p>

<ul>
<li>First make sure you format the NameNode</li>
<li>Then start the NameNode and DataNodes</li>
<li><p>Run some basic HDFS commands such as </p>

<p>hdfs dfs -put  </p></li>
</ul>

<p>and</p>

<pre><code>hdfs dfs -get
</code></pre>

<ul>
<li>Then you can start the JobTracker and TaskTracker</li>
<li>Then (and not earlier) you can try to run some MapReduce job (which uses hdfs)</li>
</ul>
