    <div class="post-text" itemprop="text">

<p>I am trying to configure and run a Hadoop Streaming job from Java (the system I'm working with wants the Hadoop jobs to be callable by Java method).</p>

<p>I did find the <code>createJob</code> method in <code>org.apache.hadoop.streaming.StreamJob</code> (<a href="http://hadoop.apache.org/common/docs/r0.20.2/api/org/apache/hadoop/streaming/StreamJob.html#createJob%28java.lang.String%5B%5D%29" rel="nofollow">http://hadoop.apache.org/common/docs/r0.20.2/api/org/apache/hadoop/streaming/StreamJob.html#createJob(java.lang.String[])</a>), but this method returns a <code>JobConf</code> object, which has been deprecated. Is there any way to create a <code>Job</code> object for streaming, or to convert the <code>JobConf</code> to a <code>Job</code>?</p>

<p>There are a few JIRA tickets about porting the Streaming code to support the new API, but to my knowledge it has yet to be made generally available</p>

<ul>
<li><a href="https://issues.apache.org/jira/browse/MAPREDUCE-1122" rel="nofollow">https://issues.apache.org/jira/browse/MAPREDUCE-1122</a></li>
<li><a href="https://issues.apache.org/jira/browse/MAPREDUCE-3619" rel="nofollow">https://issues.apache.org/jira/browse/MAPREDUCE-3619</a></li>
</ul>

<p>Note that even though the method of submitting old-api jobs using JobConf is deprecated, it's still supported in the latest 1.0.3 release of Hadoop.</p>
