<td class="answercell">
    <div class="post-text" itemprop="text">
<p>We solved this problem. </p>

<p>The root cause is that when generating the <code>java</code> command line, our Spark uses single quote('-Dxxxx') to wrap the parameters. Single quote works only in Linux. On Windows, the parameters are either not wrapped, or wrapped with double quotes("-Dxxxx"). The only way to solve this is to edit the source code of Spark and re-compile it.</p>

<p>It seems that this is currently an issue of Spark. (<a href="https://issues.apache.org/jira/browse/SPARK-5754" rel="nofollow">https://issues.apache.org/jira/browse/SPARK-5754</a>)  </p>
