<p>To my understanding, the pom dependency only fetches the thing specified in their pom like jar or war export bundle. that does not come with sources, normally. Code navigation things only work with project specific class path declarations. (<code>.classpath</code> files) of each project individually.</p>

<pre><code>    &lt;classpathentry kind="con" path="org.eclipse.jdt.launching.JRE_CONTAINER/org.eclipse.jdt.internal.debug.ui.launcher.StandardVMType/JavaSE-1.6"&gt;

            &lt;attribute name="maven.pomderived" value="true"/&gt;
