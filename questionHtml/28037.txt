    <div class="post-text" itemprop="text">
<pre><code>public static Stream&lt;String&gt; lines(Path path)
                            throws IOException
</code></pre>

<p>Read all lines from a file as a Stream. Bytes from the file are decoded into characters using the UTF-8 charset.</p>

<p>This method works as if invoking it were equivalent to evaluating the expression:</p>

<pre><code> Files.lines(path, StandardCharsets.UTF_8)
</code></pre>

<p><strong>Parameters:</strong>
path - the path to the file</p>

<p><strong>Returns:</strong> the lines from the file as a Stream</p>

<p><strong>Throws:</strong> </p>

<p><strong>IOException</strong> - if an I/O error occurs opening the file</p>

<p><strong>SecurityException</strong> - In the case of the default provider, and a security manager is installed, the checkRead method is invoked to check read access to the file.</p>

<p><strong>Since:</strong> 1.8</p>

<p><a href="http://docs.oracle.com/javase/8/docs/api/java/nio/file/Files.html#lines-java.nio.file.Path-" rel="nofollow"><code>Files.lines(Path)</code></a> expects a <code>Path</code> argument and returns a <code>Stream&lt;String&gt;</code>. 
This is Java 8, so you can use lambda expressions or method references to provide a <code>Consumer</code> argument.</p>

<pre><code>public class FixedWidthFile {

    public static void main(String JavaLatte[]) {
        Path path = Paths.get("/home/sample.txt");
        try {
            Stream&lt;String&gt; lines = Files.lines(path);
            lines.forEach(s -&gt; System.out.println(s));
        } catch (IOException ex) {

        }
    }
}
</code></pre>

<p>Reference: <a href="https://docs.oracle.com/javase/8/docs/api/java/nio/file/Files.html#lines-java.nio.file.Path-" rel="nofollow">Class Files</a></p>
