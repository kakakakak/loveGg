    <div class="post-text" itemprop="text">

<p>I'm trying to use the hbase rest API. I've read the docs here (<a href="http://wiki.apache.org/hadoop/Hbase/Stargate" rel="nofollow">http://wiki.apache.org/hadoop/Hbase/Stargate</a>), but they seem to be incomplete (doesn't cover json) and <a href="http://mail-archives.apache.org/mod_mbox/hbase-user/201402.mbox/%3CCAOEq2C5i4OH9%2BwtiwPAoX_pAp7it%2BoLAPNTBvQZWamQ05_4dcQ%40mail.gmail.com%3E" rel="nofollow">out of date</a>.</p>

<p>I've also run across <a href="https://gist.github.com/karmi/1218928" rel="nofollow">this gist</a>, which is mostly what I'm going off of.</p>

<p>Here is the (insert) request I'm trying to send:</p>

<ul>
<li>URI: <code>http://server:8070/table/row1/data</code></li>
<li>Headers: 

<ul>
<li><code>Content-Type</code> : <code>application/json</code></li>
<li><code>Accept</code> : <code>application/json</code></li>
</ul></li>
<li>HTTP verb - <code>POST</code></li>
</ul>

<p>and the json payload (where key, column, and $ are all base64 encoded values):</p>

<pre><code>{
    "Row": {

        "key": "NjQ=",
        "Cell": [{
            "column": "NjQ=",
            "$": "NjQ="
        }]
    }
}
</code></pre>

<p>Here is the error I get: </p>

<pre><code>Error 500 Can not deserialize instance of java.util.List out of START_OBJECT token
 at [Source: org.mortbay.jetty.HttpParser$Input@3da0b822; line: 2, column: 5]
 (through reference chain: org.apache.hadoop.hbase.rest.model.CellSetModel["Row"])
</code></pre>

<p>The error seems to suggest that there is something that should be an array and isn't. I've tried putting square brackets just about everywhere, and I can't get it to do anything other than change the nature of the error message slightly. </p>

<p>I had a look through the <a href="https://git-wip-us.apache.org/repos/asf?p=hbase.git;a=tree;f=hbase-rest/src/main/java/org/apache/hadoop/hbase/rest/model;hb=HEAD" rel="nofollow">source code</a> and that seems to suggest that the placement for the square brackets that I have is correct. However, as far as I can tell the entire request looks to be correct. I'm not very fluent in Java though, so maybe I'm missing something. </p>

<p>What is the proper JSON syntax for inserting a record with the hbase rest gateway?</p>

            "column": "NjQ=",
            "$": "NjQ="
        }]
    }]
}
</code></pre>

<p>Where:</p>

<ul>
<li>key - is the row key</li>
<li>column - is the column name</li>
<li>$ - is the data that you are storing in the given table, column, and row.</li>
