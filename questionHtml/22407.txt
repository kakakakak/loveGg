<p>I would suggest you downloading million songs Dataset from the following website:</p>

<p><a href="http://labrosa.ee.columbia.edu/millionsong/">http://labrosa.ee.columbia.edu/millionsong/</a></p>

<p>The best thing with Millions Songs Dataset is that you can download 1GB (about 10000 songs), 10GB, 50GB or about 300GB dataset to your Hadoop cluster and do whatever test you would want. I love using it and learn a lot using this data set. </p>

<p>To start with you can download dataset start with any one letter from A-Z, which will be range from 1GB to 20GB.. you can also use Infochimp site:</p>

<p><a href="http://www.infochimps.com/collections/million-songs">http://www.infochimps.com/collections/million-songs</a></p>

<p>In one of my following blog I showed how to download 1GB dataset and run Pig scripts:</p>

<p><a href="http://blogs.msdn.com/b/avkashchauhan/archive/2012/04/12/processing-million-songs-dataset-with-pig-scripts-on-apache-hadoop-on-windows-azure.aspx">http://blogs.msdn.com/b/avkashchauhan/archive/2012/04/12/processing-million-songs-dataset-with-pig-scripts-on-apache-hadoop-on-windows-azure.aspx</a></p>

<td class="answercell">
    <div class="post-text" itemprop="text">
<p>An article that might be of interest to you, "<a href="http://www.mappian.com/blog/hadoop/using-hadoop-to-analyze-the-full-wikipedia-dump-files-using-wikihadoop/">Using Hadoop to analyze the full Wikipedia dump files using WikiHadoop</a>". </p>

<p>If you are after <strong>Wikipedia page view statistics</strong>, then <a href="http://dumps.wikimedia.org/other/pagecounts-raw/">this</a> might help. You can download pagecount files from 2007 up until current date. Just to give an idea of the size of the files, <strong>1.9 GB for a single day</strong> (<em>here I chose 2012-05-01</em>) spread across 24 files. </p>

<p>Currently, 31 countries have sites which make available public data in various formats, <a href="http://www.data.gov/opendatasites">http://www.data.gov/opendatasites</a>. In addition, the World Bank makes available data at <a href="http://data.worldbank.org/data-catalog">http://data.worldbank.org/data-catalog</a></p>

<p>Tom White mentioned about a sample weather data set in his Book(Hadoop: the definitive guide).</p>

<p><code>http://hadoopbook.com/code.html</code></p>

<p>Data is available for more than 100 years.</p>

<p>I used <code>wget</code> in linux to pull the data. For the year 2007 itself the data size is 27 GB.</p>

<p>It is hosted as an <code>FTP</code> link. So, you can download with any FTP utility.</p>

<p><a href="ftp://ftp.ncdc.noaa.gov/pub/data/noaa/">ftp://ftp.ncdc.noaa.gov/pub/data/noaa/</a></p>

<p>For complete details please check my blog: </p>

<p><a href="http://myjourneythroughhadoop.blogspot.in/2013/07/how-to-download-weather-data-for-your.html">http://myjourneythroughhadoop.blogspot.in/2013/07/how-to-download-weather-data-for-your.html</a></p>
