<td class="answercell">
    <div class="post-text" itemprop="text">
<p>you need to add following configuration into hive-site.xml</p>

<pre><code>&lt;property&gt;
    &lt;name&gt;hive.querylog.location&lt;/name&gt;
    &lt;value&gt;$HIVE_HOME/iotmp&lt;/value&gt;
    &lt;description&gt;Location of Hive run time structured log file&lt;/description&gt;
  &lt;/property&gt;

  &lt;property&gt;
    &lt;name&gt;hive.exec.local.scratchdir&lt;/name&gt;
    &lt;value&gt;$HIVE_HOME/iotmp&lt;/value&gt;
    &lt;description&gt;Local scratch space for Hive jobs&lt;/description&gt;
  &lt;/property&gt;

  &lt;property&gt;
    &lt;name&gt;hive.downloaded.resources.dir&lt;/name&gt;
    &lt;value&gt;$HIVE_HOME/iotmp&lt;/value&gt;
    &lt;description&gt;Temporary local directory for added resources in the remote file system.&lt;/description&gt;
  &lt;/property&gt;
</code></pre>

<p>remember to replace $HIVE_HOME with your hive directory</p>

 &lt;/property&gt;



 &lt;property&gt;
   &lt;name&gt;hive.downloaded.resources.dir&lt;/name&gt;

<p>relative path URISyntaxException exception is before of you must be specifying the relative path.
I too have faced the similar issue while trying to start HMaster , the error was because i was using relative path to hdfs  for hbase.rootdir property in hbase-site.xml
on correcting the root dir path was able to start the Hmaster normally</p>

<p>path causing exception : hdfs://localhost:8020</p>

<p>correct path : hdfs://localhost:8020/hbase</p>
