<td class="answercell">
    <div class="post-text" itemprop="text">
<p>From a configuration standpoint, there's no limit I'm aware of. You can set the number of map and reduce slots to whatever you want. Practically, though, each slot has to spin up a JVM capable of running some hadoop code, which requires some amount of memory, so eventually you would run out of memory on your machine. You might also have to configure job queues cleverly in order to run a ton at the same time.</p>

<p>Now, what is possible is a very different question than what is a good idea...</p>
