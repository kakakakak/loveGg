<td class="answercell">
    <div class="post-text" itemprop="text">
<p>In addition to aasoj's answer, I would like to make a point here that the output from the mapper will be fed as input to reducer. So, in your reducer class the input key value types are 'Text' and 'IntWritable' where as the output key values of your mapper class are 'Text' and 'Text'.</p>

<p>Try to change the input key values of reducer same as the mapper's output types which looks like as below:</p>

<p>public  class DemoMapper extends Mapper</p>

<p>public class DemoReducer extends Reducer</p>

<p>Apart from the above everything looks for me. </p>
