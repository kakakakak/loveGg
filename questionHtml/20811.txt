<p>Is there a standard or de facto standard GUI administration panel for Redis? I'd like to see general health and status of my Redis instances through a web interface. Advanced stuff such as access to logs, trends on memory usage, etc. would be nice but not necessary. I'm running Redis on a Hadoop cluster, in which I enjoy having pages for the JobTracker, NameNode, Ganglia, etc.</p>

<p>There are a few out there, but at first glance they don't seem ready for prime time.</p>

<td class="answercell">
    <div class="post-text" itemprop="text">
<p>Give a try to <a href="https://redsmin.com" rel="nofollow">Redsmin</a> (<a href="http://stackoverflow.com/a/12316197/745121">long description here</a>). Its a <strong>fully online</strong> GUI for Redis that provides a lot of features.</p>

<p>It currently offers:</p>

<ul>
<li><strong>No installation</strong> and cross-platform</li>
<li>Multiple <strong>database management</strong> (with direct or proxied access)</li>
<li>Batch operation over multiple key that match a pattern</li>
<li><strong>Value editor</strong></li>
<li>Online <strong>configuration</strong></li>
<li>JavaScript API directly accessible from the browser console for light data processing</li>
<li><a href="http://www.youtube.com/watch?v=JA9UvdgJ_GI" rel="nofollow"><strong>Monitoring</strong> features</a></li>
<li>Online <strong>terminal</strong> with auto-completion and <strong>inline-documentation</strong></li>
<li>Real-time <strong>data-visualization</strong></li>
<li><strong>Monitoring</strong> (and alerting soon)</li>
</ul>

<a href="https://redsmin.com" rel="nofollow"><img src="http://i.stack.imgur.com/bU2V7.jpg" alt="enter image description here"></a></p>
