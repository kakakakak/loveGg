<td class="answercell">
    <div class="post-text" itemprop="text">
<p>You have not defined "highsal" alias when trying to run STORE command. 
Pig do not store any alias previous session. you have to execute all your command in one session or write a pig script and invoke it.</p>

<p>Try like :</p>

<pre><code>grunt&gt;salaries= load 'salaries' using PigStorage(',') As (gender, age,salary,zip);
grunt&gt; salaries= load 'salaries' using PigStorage(',') As (gender:chararray,age:int,salary:double,zip:long);
grunt&gt;highsal= filter salaries by salary &gt; 75000;
grunt&gt;STORE highsal INTO 'file';
</code></pre>

<p>This will store the "highsal" content in a file name 'file/part-x-xxxxx' on user's HDFS directory. You can also provide HDFS absolute directory path instead of <code>'file'</code> if you want to wish to store data in directory other than users home directory</p>

<p>Hope this helps</p>
