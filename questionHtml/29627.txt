<td class="answercell">
    <div class="post-text" itemprop="text">
<p>There are three ways to describe a table in Hive.</p>

<p>1) To see table primary info of Hive table, use <strong>describe table_name;</strong> command </p>

<p>2) To see more detailed information about the table, use <strong>describe extended table_name;</strong> command</p>

<p>3) To see code in a clean manner use <strong>describe formatted table_name;</strong> command to see all information. also describe all details in a clean manner.</p>

<p>Resource: <a href="http://www.bigdataanalyst.in/hive-interview-questions/" rel="nofollow">Hive interview tips</a></p>
