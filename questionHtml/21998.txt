<p>This is just a tip for others facing similar issue:<br>
please make sure that you set the values first and declare a job.</p>

<p>For example:</p>

<pre><code>Configuration conf = new Configuration(); 
conf.set("a","2"); 
conf.set("inputpath",args[0]);
//Must be set before the below line:
Job myjob = new Job(conf); 
</code></pre>
