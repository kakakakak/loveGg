    <div class="post-text" itemprop="text">
<p><strong>Single node cluster</strong> : By default, Hadoop is configured to run in a non-distributed or standalone mode, as a single Java process. There are no daemons running and everything runs in a single JVM instance. HDFS is not used.</p>

<p><strong>Pseudo-distributed or multi-node cluster</strong>: The Hadoop daemons run on a local machine, thus simulating a cluster on a small scale. Different Hadoop daemons run in different JVM instances, but on a single machine. HDFS is used instead of local FS</p>
