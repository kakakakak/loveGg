<td class="answercell">
    <div class="post-text" itemprop="text">
<p>I think you should specify more executors while submitting the application. For example:</p>

<pre><code>spark-submit --master local[4] your_file.py
</code></pre>

<p>From <em>Learning Spark</em> chapter 10:</p>

<blockquote>
  <p>Do not run Spark Streaming programs locally with <code>master</code> configured as
  <code>local</code> or <code>local[1]</code>. This allocates only one CPU for tasks and if a
  receiver is running on it, there is no resource left to process the
  received data. Use at least <code>local[2]</code> to have more cores.</p>
</blockquote>
