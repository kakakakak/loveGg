<td class="answercell">
    <div class="post-text" itemprop="text">
<p>If you want to retrieve <strong>simple data</strong> as lists or rows with some columns the option #2 is the good one, you can read below a set of advantages and disadvantages:</p>

<p><strong>Cons</strong></p>

<ul>
<li><strong>Less bandwidth</strong> needed because json needs more syntax characters to keep the format while csv is as simple as use a character separator</li>
<li><strong>Process data is faster</strong> because only needs to split by the separator character while JSON needs to interpret the syntax</li>
<li>Big data technology as <strong>Hadoop have an integrated parse for CSV format</strong> while needs an specific function for parse Json (for example using Hive language).</li>
</ul>

<p><strong>Pros</strong></p>

<ul>
<li>Unstructured data and more difficult to be read by humans</li>
<li>You have to take care with separator character cannot appear in data fields.</li>
</ul>

<p>If the data will contains <strong>complex data</strong> as tupla, arrays and structures JSON is better because:</p>

<ul>
<li>Keeps a clear and <strong>structured format</strong></li>
<li><strong>Doesn't repeat data</strong> to reference it because one label could contain multiple data.</li>
</ul>
