<p>I have problem to use maven from ubuntu server 13.10 64bit.
I am getting following error (even when I type <strong>mvn -version</strong>):</p>

<pre><code>Error: JAVA_HOME is not defined correctly.
     We cannot execute /usr/lib/jvm/jdk//bin/java
</code></pre>

<p>But when I type <strong>java -version</strong> I get:</p>

<pre><code>java version "1.7.0_51"
Java(TM) SE Runtime Environment (build 1.7.0_51-b13)
Java HotSpot(TM) 64-Bit Server VM (build 24.51-b03, mixed mode)
</code></pre>

<p>I have no idea where can be the problem, especially because I have used maven without any problems before I've installed hadoop 2.2.</p>

<p>Thank you in advance for any help. </p>
