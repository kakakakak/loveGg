<td class="answercell">
    <div class="post-text" itemprop="text">
<p>Are you looking for a distributed file system at the same time? If so I suggest using Apache Hadoop's <a href="http://wiki.apache.org/hadoop/HDFS" rel="nofollow">HDFS</a> and the <a href="http://hadoop.apache.org/docs/current/hadoop-project-dist/hadoop-hdfs/WebHDFS.html" rel="nofollow">WebHDFS REST API</a> to access the file system.</p>

<p>How ever I am not sure whether this can be deployed as an extension to Apache or any other web server :-( Just wanted to share this idea, if you are looking for a distributed file system with guaranteed reliability etc.</p>
