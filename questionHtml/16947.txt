<td class="answercell">
    <div class="post-text" itemprop="text">
<p>Yes, it is a compatibility issue.</p>

<ul>
<li>in Hadoop 1.x: <a href="http://hadoop.apache.org/docs/r1.2.1/api/org/apache/hadoop/mapreduce/JobContext.html">JobContext is a class</a></li>
<li>in Hadoop 2.x: <a href="http://hadoop.apache.org/docs/current/api/org/apache/hadoop/mapred/JobContext.html">JobContext is an interface</a></li>
</ul>

<p>You can see that Hadoop 2 is not listed as compatible with Pig 0.12:</p>

<pre><code>Requirements
Mandatory

Unix and Windows users need the following:

Hadoop 0.20.2, 020.203, 020.204, 0.20.205, 1.0.0, 1.0.1, or 0.23.0, 0.23.1 
</code></pre>

<p><a href="http://pig.apache.org/docs/r0.12.0/start.html">http://pig.apache.org/docs/r0.12.0/start.html</a></p>
