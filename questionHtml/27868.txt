<td class="answercell">
    <div class="post-text" itemprop="text">
<p>1) Install &amp; configure MySQL first. Create database in MySQL </p>

<p>2) <code>sqoop import --connect jdbc:mysql://localhost/databasename --username $USER_NAME --password $PASSWORD$ --table tablename --m 1</code> command will import data.</p>

<p>e.g.</p>

<p>sqoop import --connect jdbc:mysql://localhost/testDb --username root --password hadoop123 --table student --m 1</p>

<p>In above command, values of various parameters <code>database:‘testDb’</code> , <code>username: ‘root’</code>, <code>password: ‘hadoop123’</code>, and <code>table student</code>.</p>

<p>Have a look at this <a href="https://archanaschangale.wordpress.com/2013/09/18/sqoopimporting-data-from-mysql-into-hdfs/" rel="nofollow">article 1</a> and <a href="https://sqoop.apache.org/docs/1.4.3/SqoopUserGuide.html" rel="nofollow">article 2</a> for better understanding in step-by-step manner</p>
