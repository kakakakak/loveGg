<p>You can use TeraSort, the benchmarking test packaged with Hadoop. It sorts terabytes of data, and is used to stress test new Hadoop clusters. It's part of the <code>hadoop-x.y.z-examples.jar</code> file that comes with a Hadoop install.</p>

<p>To use it, generate data into HDFS using Teragen, then run Terasort.</p>
