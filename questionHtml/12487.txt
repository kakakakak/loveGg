<td class="answercell">
    <div class="post-text" itemprop="text">
<p>Well it cannot resolve the host SRV101004013. The one question is did you write this host to some Kafka or ZK config? Can you resolve the hostname using <code>nslookup</code> or simple <code>ping</code>? If yes do you have the host in /etc/hosts? Because <a href="http://stackoverflow.com/questions/1881546/inetaddress-getlocalhost-throws-unknownhostexception">InetAddress.getLocalHost() ignores the /etc/resolv.conf but only looks at the /etc/hosts file</a>.</p>
