<p>I ran a recursive map/reduce program. Something went wrong and it nearly consumes all the disk space available in C drive. So i closed the <strong>resource manager</strong>, <strong>node manager</strong>, <strong>Name Node</strong>, <strong>data node</strong> consoles. 
Now i have a C drive which is almost full and i don't know how to empty the disk space and make my C drive as it was before. What should i do now. Any help is appreciated.
Here is the code </p>

{
    private Text word = new Text();

    public void CandidatesGenRecursion(Vector&lt;String&gt; in, Vector&lt;String&gt; out,

        int i,size;

                out.add(in.get(i));

                out.add(in.get(i));

                int init=1;
                StringBuffer current = new StringBuffer();
                for(String s:out)
                {   
                    if(init==1){
                        current.append(s);  
                        init=0;
                    } else {
                        current.append(" ");
                        current.append(s);  
                    }
                }


                word.set(current.toString());
                count.set(Integer.toString(1));
                try {
                    context.write(word, count);
                } catch (InterruptedException e) {

            }
            if(i &lt; length-1) {
                CandidatesGenRecursion(in, out, length,level+1,i+1, context);

            size = out.size();
            if(size&gt;0){
                out.remove(size-1);
            }
        }

    }

    @Override
    public void map(LongWritable key,Text value,Context context) throws IOException
    {
        String line = value.toString();
        StringTokenizer tokenizer = new StringTokenizer(line);
        String[] token=new String[2];
       int i=0;
        while(tokenizer.hasMoreTokens()){
           token[i]= tokenizer.nextToken();
           ++i;
        }

        StringTokenizer urlToken = new StringTokenizer(token[1],",");


        Vector&lt;String&gt; lst = new Vector&lt;String&gt;(); 
        int loop=0;
        while (urlToken.hasMoreTokens()) {

            String str = urlToken.nextToken();

                lst.add(str);
                loop++;

        }


        Vector&lt;String&gt; combinations = new Vector&lt;String&gt;(); 

        if(!lst.isEmpty()) {
            CandidatesGenRecursion(lst, combinations, loop,0,0, context);

public static class CandidateGenReduce extends Reducer&lt;Text, IntWritable, Text, IntWritable&gt;
{

    public void reduce(Text key,Iterator&lt;IntWritable&gt; values,Context context) throws IOException
    {

        int sum = 0;

        try {               
            context.write(key, new IntWritable(sum));

    }

<td class="answercell">
    <div class="post-text" itemprop="text">
<p>Check in your HDFS your job output path and delete the contents.</p>

<p>List contents:</p>

<pre><code>$ sudo -u hdfs hadoop fs -ls [YourJobOutputPath]
</code></pre>

<p>Disk used:</p>

<pre><code>$ sudo -u hdfs hadoop fs -du -h [YourJobOutputPath]
</code></pre>

<p>Delete contents (be careful!, it's recursive):</p>

<pre><code>$ sudo -u hdfs hadoop fs -rm -R [YourJobOutputPath]
</code></pre>
