    <div class="post-text" itemprop="text">

<p>I start with the MRAppMaster.java code.I want to debug into it to see the principle of a yarn application.</p>

<p>I import the whole compiled hadoop code in my Eclipse.</p>

<p>I run hadoop all in my local Linux.</p>

<pre><code>4504 ResourceManager 
4066 DataNode 
4761 NodeManager 
5068 JobHistoryServer 
4357 SecondaryNameNode 
3833 NameNode 
5127 Jps
</code></pre>

<p>In hadoop-env.sh:</p>

<pre><code>HADOOP_OPTS="$HADOOP_OPTS -Xdebug -Xrunjdwp:transport=dt_socket,server=y,address=9000"
</code></pre>

<p>And then I run：</p>

<pre><code>bin/hadoop jar path/to/hadoop-mapreduce-examples-2.2.0.jar wordcount /wordcount /output
</code></pre>

<p>Then I set breakpoint in MRAppMaster.java and RunJar.java . Then start debug in Eclipse.</p>

<p>It only stop at :</p>

<pre><code>Thread [main] (Suspended (breakpoint at line 342 in JobSubmitter)) 
JobSubmitter.submitJobInternal(Job, Cluster) line: 342 
...
RunJar.main(String[]) line: 212
</code></pre>

<p>How can I debug into MRAppMaster.java?</p>
