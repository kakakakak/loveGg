<li>Point your web browser to <strong>HDFS WEBUI(namenode_machine:50070)</strong>, browse to the file you intend to copy, scroll down the page and click on <strong>download the file</strong>.</li>

<td class="answercell">
    <div class="post-text" itemprop="text">
<p>In order to copy files from HDFS to the local file system the following command could be run:</p>

<p><code>hadoop dfs -copyToLocal &lt;input&gt; &lt;output&gt;</code></p>

<ul>
<li><code>&lt;input&gt;</code>: the HDFS directory path (e.g /mydata) that you want to copy</li>
<li><code>&lt;output&gt;</code>: the destination directory path (e.g. ~/Documents)</li>
</ul>
