<td class="answercell">
    <div class="post-text" itemprop="text">
<p>The problem is that the class that you're using for <code>map()</code> output / <code>reduce()</code> input, <code>Weather</code> does not implement <code>Writable</code>.  This will prevent the default <code>SerializationFactory</code> from being able to process your values.</p>

<p>The underlying conceptual problem is that Hadoop does not know how to serialize your data type to disc and read it back.  That is a mandatory step, because the data has to be persisted before it can be moved from the map task to a reducer (the two can run on separate nodes, in general).</p>

<p>So what you want to do is implement <code>Writable</code> and add serialization routines in your custom data type.</p>
