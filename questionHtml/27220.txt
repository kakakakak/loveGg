<p>I am trying to implement a statistical formula that requires comparing a datapoint with all other possible datapoints. For example my dataset is something like:</p>

<pre><code>10.22
15.77
16.55
9.88
</code></pre>

<p>I need to go through this file like:</p>

<pre><code>for (i=0;i&lt; data.length();i++)
   for (j=0;j&lt; data.length();j++)
     Sum +=(data[i] + data[j])
</code></pre>

<p>Basically when i get each line through my map function, i need to execute some instructions on the rest of the file in the reducer like in a nested for loop. 
Now i have tried using the distributedCache, some form of ChainMapper but to no avail. Any idea of how i can go about doing this would be really appreciated. Even an out of the box way will be helpful.</p>
