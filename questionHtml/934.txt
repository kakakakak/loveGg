    <div class="post-text" itemprop="text">
<p>I would think that the one live node appearing there is the master and not the slave. I guess the problem is in your <code>/etc/hosts</code> file. Did you place the slave there too?</p>

<p>Moreover, if you followed the standard installation steps, like <a href="http://www.michael-noll.com/tutorials/running-hadoop-on-ubuntu-linux-multi-node-cluster/" rel="nofollow">these</a>, then the "slaves" file should be under <code>$HADOOP_HOME/conf/</code> and not <code>etc/hadoop/</code>. <code>/usr/local/hadoop</code> is more commonly used as <code>$HADOOP_HOME</code>. Make sure you covered all the steps of the installation correctly before proceeding, otherwise more problems will rise. </p>
