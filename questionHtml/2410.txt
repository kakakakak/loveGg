<p>You can specify ',' in PigStorage Class to read CSV file.</p>

<p>Query looks like : </p>

<pre><code>grunt&gt; inputfile= load '/training/pig/datetemp.txt' using PigStorage(',') As (EventID: chararray,eventdate: chararray,count:int);

grunt&gt; dump inputfile;
</code></pre>

<p>And make sure that you have file '/training/pig/datetemp.txt' on HDFS.
To test run : <code>hadoop fs -ls /training/pig/datetemp.txt</code></p>
