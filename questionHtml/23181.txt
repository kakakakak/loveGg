<td class="answercell">
    <div class="post-text" itemprop="text">
<p>I had the same problem, this is how I got it to work.</p>

<p>I first tried replacing ": " by ",":</p>

<pre><code>sed -i 's/: /,/' links-simple-sorted.txt
</code></pre>

<p>since that didn' work, I looked up the documentation, and it seems the file has to be reformatted: Each line has to be replaced by 1+x lines, each starting with the user name, followed by one link per line.:</p>

<pre><code>awk -F, -v OFS="," '{ user = $1; split($2, links, " "); for (link in links) { print user,links[link]; } }' links-simple-sorted.txt &gt; input.txt
</code></pre>

<p>then I uploaded the new file:</p>

<pre><code>bin/hadoop fs -put input.txt input/input.txt
</code></pre>

<p>Now the example is running with the above command <code>bin/hadoop jar ...</code>.</p>
