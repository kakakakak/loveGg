<td class="answercell">
    <div class="post-text" itemprop="text">
<p>I had a similar problem with integration tests I need to run on Hadoop. Our integration tests are located in the <code>test</code> folder of a separate integration test module, so what is required is a <code>test-jar-with-dependencies</code> to make our life easier.</p>

<p>I'm using the <a href="http://maven.apache.org/plugins/maven-assembly-plugin/">assembly plugin</a> as mentioned by Michael-O. My assembly descriptor is located in <code>src/main/assembly/test-jar-with-dependencies.xml</code> and is a modification of the standard <a href="http://maven.apache.org/plugins/maven-assembly-plugin/descriptor-refs.html#jar-with-dependencies"><code>jar-with-dependencies</code></a> descriptor that's part of the plugin:</p>

<pre><code>&lt;assembly xmlns="http://maven.apache.org/plugins/maven-assembly-plugin/assembly/1.1.0"
      xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
      xsi:schemaLocation="http://maven.apache.org/plugins/maven-assembly-plugin/assembly/1.1.0 http://maven.apache.org/xsd/assembly-1.1.0.xsd"&gt;
    &lt;id&gt;test-jar-with-dependencies&lt;/id&gt;
    &lt;formats&gt;
        &lt;format&gt;jar&lt;/format&gt;
    &lt;/formats&gt;
    &lt;includeBaseDirectory&gt;false&lt;/includeBaseDirectory&gt;
    &lt;dependencySets&gt;
        &lt;dependencySet&gt;
            &lt;outputDirectory&gt;/&lt;/outputDirectory&gt;
            &lt;useProjectArtifact&gt;true&lt;/useProjectArtifact&gt;
            &lt;!-- we're creating the test-jar as an attachement --&gt;
            &lt;useProjectAttachments&gt;true&lt;/useProjectAttachments&gt;
            &lt;unpack&gt;true&lt;/unpack&gt;
            &lt;scope&gt;test&lt;/scope&gt;
        &lt;/dependencySet&gt;
    &lt;/dependencySets&gt;
&lt;/assembly&gt;
</code></pre>

<p>This assembly relies on the <code>test-jar</code> being created as part of the module build. So I added the following to the module's <code>pom.xml</code>:</p>

<pre><code>&lt;!-- create a complete jar for testing in other environments --&gt;
&lt;plugin&gt;
    &lt;groupId&gt;org.apache.maven.plugins&lt;/groupId&gt;                    
    &lt;artifactId&gt;maven-jar-plugin&lt;/artifactId&gt;
    &lt;executions&gt;
        &lt;execution&gt;
            &lt;goals&gt;
                &lt;goal&gt;test-jar&lt;/goal&gt;
            &lt;/goals&gt;
        &lt;/execution&gt;
    &lt;/executions&gt;
&lt;/plugin&gt;
&lt;plugin&gt;
    &lt;groupId&gt;org.apache.maven.plugins&lt;/groupId&gt;
    &lt;artifactId&gt;maven-assembly-plugin&lt;/artifactId&gt;
    &lt;configuration&gt;
        &lt;descriptors&gt;
            &lt;descriptor&gt;src/main/assembly/test-jar-with-dependencies.xml&lt;/descriptor&gt;
        &lt;/descriptors&gt;
    &lt;/configuration&gt;
    &lt;executions&gt;
        &lt;execution&gt;
            &lt;phase&gt;package&lt;/phase&gt;
            &lt;goals&gt;
                &lt;goal&gt;single&lt;/goal&gt;
            &lt;/goals&gt;
        &lt;/execution&gt;
    &lt;/executions&gt;
&lt;/plugin&gt;
</code></pre>
