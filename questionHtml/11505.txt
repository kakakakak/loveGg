<td class="answercell">
    <div class="post-text" itemprop="text">
<p>You could write your own job to migrate the data. But Sqoop would be more convenient. To do that you have to <a href="http://apache.osuosl.org/sqoop/1.4.3/sqoop-1.4.3.tar.gz" rel="nofollow">download</a> Sqoop and the appropriate connector, Microsoft SQL Server Connector for Apache Hadoop in your case. You can download it from <a href="http://www.microsoft.com/en-us/download/details.aspx?id=27584" rel="nofollow">here</a>.Please go through the <a href="http://sqoop.apache.org/docs/1.4.2/SqoopUserGuide.html" rel="nofollow">Sqoop user guide</a>. It contains all the information in proper detail. </p>

<p>And Hive does support ODBC. You can find more on this at this <a href="https://cwiki.apache.org/Hive/hiveodbc.html" rel="nofollow">page</a>.</p>
