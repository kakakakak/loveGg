    <div class="post-text" itemprop="text">

<p>I'm looking into tuning MapR Hadoop via Ansible templates.</p>

<p>It is easy enough to tune something to the number of CPU threads found on a system.   For example, to set reduce tasks to 1/4 of threads:</p>

<pre><code>&lt;name&gt;mapred.tasktracker.reduce.tasks.maximum&lt;/name&gt;
&lt;value&gt;{{ (ansible_processor_vcpus / 4)|int }}&lt;/value&gt;
</code></pre>

<p>One resource suggests that the number of map / reduce tasks should be scaled to the number of <strong>disks</strong> on the system.  I don't see any comparable variable for that.</p>

<p>There is an array of <code>ansible_devices</code> with sda, sdb, &amp;c.  Perhaps I can count that?  Perhaps apply a filter so I'm only counting disks which are available to Hadoop?</p>
