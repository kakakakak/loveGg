<td class="answercell">
    <div class="post-text" itemprop="text">
<p>I find <a href="http://mahout.apache.org/" rel="nofollow">Apache Mahout</a> an excellent option to build Machine Learning Based applications. It supports <strong><em>clustering, classfication and batch based collaborative filtering</em></strong> using a number of standard algorithms. <strong>Apache Mahout In Action</strong> is a good book to get started with Machine Learning and learn about <strong><em>clustering, recommendations, classification</em></strong> algorithms using the Mahout Library.</p>

<p>The great thing about this library is that it is highly scalable and the examples provided in the release give useful insights into using it to build full fledged applications. It <strong><em>harnesses the capability of Apache Hadoop and Map Reduce</em></strong> in order to scale the data sets on which the different algorithms run and achieve great <strong><em>performance by doing distributed computing</em></strong>.</p>
