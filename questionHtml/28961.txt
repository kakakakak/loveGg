    <div class="post-text" itemprop="text">
<p>Since you said you were new to hadoop, do you know the basic idea of <code>Mapper</code> and <code>Reducer</code>? Both of them have KEY_IN_CLASS, VALUE_IN_CLASS, KEY_OUT_CLASS, VALUE_OUT_CLASS, so in your case, you can simple use mapper to do the convert, </p>

<p>for KEY_IN_CLASS, you can use the default <code>LongWritable</code>, </p>

<p>VALUE_IN_CLASS you need to use <code>Text</code>, since <code>Text</code> class deals with text input. </p>

<p>For KEY_OUT_CLASS, you can use <code>NullWritable</code>, it's a null key if you don't have a specific key. </p>

<p>For VALUE_OUT_CLASS, use <code>SequenceFileOutputFormat</code>.</p>

<p>I believe in order to use <code>SequenceFileOutputFormat</code>, you need to tell SequenceFileOutputFormat what key class and value class you use.</p>
