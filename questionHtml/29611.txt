    <div class="post-text" itemprop="text">

<p>Is there any way to include a call to write.csv within a string of piped dplyr functions?</p>

<pre><code>library(dplyr)
mtcars %&gt;% 
  filter(cyl == 4) %&gt;% 
  write.csv(file = "firstStage.csv") %&gt;% 
  group_by(carb) %&gt;% 
  summarise(hp.sum = sum(hp)) %&gt;%
  write.csv(file = "secondStage.csv")
</code></pre>

<p>I could create my own function:</p>

<pre><code>csv2go &lt;- function(x, ...) {
  write.csv(x, ...)
  x
}
</code></pre>

<p>but was wondering if there is a something that does this in base or dplyr. Maybe the <code>write_csv()</code> function in <code>library(readr)</code> could adopt this option?</p>

    <div class="post-text" itemprop="text">
<p>Very simple!!!  All you need to do is to add a single character "T" to convert the regular pipe operator %>% to the Tee-pipe operator %T>% like this:</p>

<pre><code>library(dplyr)
mtcars %&gt;% 
   filter(cyl == 4) %T&gt;%                  # &lt;== Tee-pipe operator 
   write.csv(file = "firstStage.csv") %&gt;% 
   group_by(carb) %&gt;% 
   summarise(hp.sum = sum(hp)) %&gt;%
   write.csv(file = "secondStage.csv") 
</code></pre>

<p>%T>% is your new friend</p>
