    <div class="post-text" itemprop="text">

<p>I have uploaded a Directory to hadoop cluster that is having "," in its name
like "MyDir, Name" when I am trying to delete this Directory by using rmr hadoop shell command
as following</p>

<pre><code>hadoop dfs -rmr hdfs://host:port/Navi/MyDir, Name
</code></pre>

<p>I'm getting the following messages
rmr: cannot remove hdfs://host:port/Navi/MyDir,: No such file or directory.
rmr: cannot remove Name: No such file or directory.</p>

<p>However I have successfully deleted other Directories from the same location, using the same command
i.e. </p>

<pre><code>hadoop dfs -rmr hdfs://host:port/dir_path
</code></pre>

<p>any solutions to delete such kind of Directories.</p>
