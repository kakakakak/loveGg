<p>I want to use LZO compression in MapReduce, but am getting an error when I run my MapReduce job. I am using Ubuntu with a Java program. I am only trying to run this on my local machine. My initial error is</p>

<pre><code>ERROR lzo.GPLNativeCodeLoader: Could not load native gpl library
</code></pre>

<p>and down the line</p>

<pre><code>ERROR lzo.LzoCodec: Cannot load native-lzo without native-hadoop
</code></pre>

<p>and then</p>

<pre><code>java.lang.RuntimeException: native-lzo library not available
</code></pre>

<p>I have followed a number of online and textual instructions on how to download and configure files for use with LZO compression. Here you can see my hadoop-lzo jar file in the <code>lib</code> folder</p>

<p><a href="http://i.stack.imgur.com/UyL6E.png" rel="nofollow"><img src="http://i.stack.imgur.com/UyL6E.png" alt="enter image description here"></a></p>

<p>I have changed my configurations as such. Here is my core-site.xml</p>

<pre><code>&lt;configuration&gt;
&lt;property&gt;

    &lt;value&gt;hdfs://localhost:9000&lt;/value&gt;

    &lt;name&gt;io.compression.codecs&lt;/name&gt;

&lt;/property&gt;
&lt;property&gt;

<pre><code>&lt;configuration&gt;

export JAVA_LIBRARY=/home/hadoop/lib/native/lib/lib
</code></pre>

<p>If you are interested in what is in <code>/home/hadoop/lib/native/lib/lib</code></p>

<p><a href="http://i.stack.imgur.com/Niozp.png" rel="nofollow"><img src="http://i.stack.imgur.com/Niozp.png" alt="enter image description here"></a></p>

<p>For what it's worth, here is my Driver class that does all of the compressing</p>

<pre><code>//import java.net.URI;

import org.apache.hadoop.conf.Configuration;
import org.apache.hadoop.conf.Configured;
//import org.apache.hadoop.filecache.DistributedCache;
import org.apache.hadoop.fs.Path;
import org.apache.hadoop.io.IntWritable;
import org.apache.hadoop.io.NullWritable;
import org.apache.hadoop.io.Text;
//import org.apache.hadoop.io.compress.CompressionCodec;
import com.hadoop.compression.lzo.LzopCodec;
import org.apache.hadoop.mapreduce.Job;
import org.apache.hadoop.mapreduce.lib.input.FileInputFormat;
import org.apache.hadoop.mapreduce.lib.output.FileOutputFormat;
//import org.apache.hadoop.mapreduce.lib.output.SequenceFileOutputFormat;
import org.apache.hadoop.util.Tool;
import org.apache.hadoop.util.ToolRunner;

public class FuzzyJoinDriver extends Configured implements Tool{

public static void main(String[] args) throws Exception {     
    int exitCode = ToolRunner.run(new Configuration(), new  FuzzyJoinDriver(),args);
System.exit(exitCode);
}
@Override
public int run(String[] args) throws Exception {
    if (args.length != 2) {
          System.err.println("Usage: FuzzyJoinDriver &lt;input path&gt; &lt;output path&gt;");
          System.exit(-1);
        }

        Configuration conf = new Configuration();
        //Used to compress map output
        //conf.setBoolean("mapred.compres.map.output", true);
        //conf.setClass("mapred.map.output.compression.code", GzipCodec.class, CompressionCodec.class);
        Job job = new Job(conf);
        job.setJarByClass(FuzzyJoinDriver.class);
        job.setJobName("Fuzzy Join");

        //Distributed Cache
        //DistributedCache.addCacheFile(new URI("/cache/ReducerCount.txt"),  job.getConfiguration());


        FileInputFormat.addInputPath(job, new Path(args[0]));
        FileOutputFormat.setOutputPath(job, new Path(args[1]));

        job.setMapperClass(FuzzyJoinMapper.class);


        job.setReducerClass(FuzzyJoinReducer.class);
        job.setMapOutputKeyClass(IntWritable.class);
        job.setMapOutputValueClass(RelationTuple.class);


        job.setPartitionerClass(JoinKeyPartitioner.class);


        job.setOutputKeyClass(NullWritable.class);
        job.setOutputValueClass(Text.class);

        //Used to compress output from reducer
        FileOutputFormat.setCompressOutput(job,  true);
        FileOutputFormat.setOutputCompressorClass(job, LzopCodec.class);
        //SequenceFileOutputFormat.setOutputCompressionType(job, org.apache.hadoop.io.SequenceFile.CompressionType.BLOCK);

        return(job.waitForCompletion(true) ? 0 : 1);
    }
}
</code></pre>

<td class="answercell">
    <div class="post-text" itemprop="text">
<p>You need to set the following properties in your mapred-site.xml:</p>

<pre><code>&lt;property&gt;
    &lt;name&gt;mapred.compress.map.output&lt;/name&gt;
    &lt;value&gt;true&lt;/value&gt;
&lt;/property&gt;

&lt;property&gt;
    &lt;name&gt;mapred.map.output.compression.codec&lt;/name&gt;
    &lt;value&gt;org.apache.hadoop.io.compress.LzoCodec&lt;/value&gt;
&lt;/property&gt;
</code></pre>

<p>If you want to do it on a job per job basis, you could also directly implement that in your code in 1 of the following ways:</p>

<pre><code>conf.set("mapred.compress.map.output", "true")
conf.set("mapred.map.output.compression.codec", "org.apache.hadoop.io.compress.LzoCodec");
</code></pre>

<p>or</p>

<pre><code>jobConf.setMapOutputCompressorClass(LzoCodec.class);
</code></pre>
