    <div class="post-text" itemprop="text">
<p>You need to put hbase-site.xml file of your cluster in classpath of your java application. <br>
Also, you need to remove the code that is setting configuration object. That is, these line:</p>

<pre><code>config.set("hbase.zookeeper.quorum","ip_address_of_remotemachine");
config.set("hbase.zookeeper.property.clientPort","2181");
config.set("hbase.master", "hdfs://remote_ip:2181/hbase");
</code></pre>
