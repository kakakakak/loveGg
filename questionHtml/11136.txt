<td class="answercell">
    <div class="post-text" itemprop="text">
<p>The method nextKeyValue() will be used by each mapper, to iterate between all split records.</p>

<p>The class NLinesRecordReader defines that each record has 3 lines.</p>

<pre><code>private final int NLINESTOPROCESS = 3;
</code></pre>

<p>The main role for the loop in nextKeyValue() is to get for each record 3 lines. The record will be used as an input value on the map() method. </p>

<p>Whenever a new data is required two things happens. The first question which is asked to the reader is </p>

<blockquote>
  <p>DO YOU HAVE ANY DATA ???</p>
</blockquote>

<p>If the reader replies yes then the caller can get the data from the <strong>getCurrentValue</strong> method. </p>

<p>Now the nextKeyValue method does this task, it just answers the question DO YOU HAVE ANY DATA LEFT TO GIVE ME ?</p>

<p>I am not able to access the link due to firewall issues but a sample implementation which i used is</p>

<pre><code>HashMap&lt;Integer, Invoice&gt; allData= new HashMap&lt;Integer, Invoice&gt;();

    @Override
public boolean nextKeyValue() throws IOException, InterruptedException {
    if(key == null) {
        this.key = new LongWritable();
    }
    this.key.set(startPos);

    if(value == null) {
        this.value = new Invoice();
    }
    if(startPos &gt;= endPos) {
        key = null;
        value = null;
        return false;
    } else {
        this.value = allData.get(startPos);
        startPos = startPos + 1;
        return true;
    }
}
</code></pre>

<p>Here <strong>Invoice</strong> is just a POJO. and in the initialize method i did nothing but parse the whole document and store in a hashmap. 
In the nextKeyValue method check if the next key exsists if it does return the corresponding value else return the key does not exsist.</p>

    if (value == null) {
        value = new Text();

    value.clear();
    final Text endline = new Text("\n");
