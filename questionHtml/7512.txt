<p>after several try installing Lzo compression for hadoop, I need help because I have really no idea why it doesn't work.</p>

<p><code>13/07/03 19:52:23 INFO lzo.GPLNativeCodeLoader: Loaded native gpl library</code>
<code>13/07/03 19:52:23 WARN lzo.LzoCompressor: java.lang.NoSuchFieldError: workingMemoryBuf</code></p>

<p><code>13/07/03 19:52:23 ERROR lzo.LzoCodec: Failed to load/initialize native-lzo library</code></p>

<p>The real problem is that the lzo test works on the namenode :</p>

<p><code>13/07/03 18:55:47 INFO lzo.GPLNativeCodeLoader: Loaded native gpl library</code>

<p>I've try setting several path in haddop-env.sh but there seems to be no right solution...

I've try  sudhirvn.blogspot.fr/2010/08/hadoop-lzo-installation-errors-and.html but removing all Lzo and gplcompression libraries and then making a nez install was not better at all.</p>

<p><code>CLASSPATH=/usr/lib/hadoop/hadoop-core-1.0.4.jar CFLAGS=-m64 CXXFLAGS=-m64 ant compile-native tar</code></p>
