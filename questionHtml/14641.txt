<td class="answercell">
    <div class="post-text" itemprop="text">
<p>The registry key 
<code>HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\MSBuild\ToolsVersions\14.0@VCTargetsPath</code>
found in the CMakeError.log above belongs to Visual Studio 2015. However the output of MSBuild looks like Visual Studio 2010. You probably had VS2015 installed and then uninstalled. The uninstaller of VS2015 has a bug, as it does not clean up the registry properly.</p>

<p>On a 64 bit machine, this key is in fact located at <code>HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\MSBuild\ToolsVersions</code>. If there is a subkey named <code>14.0</code> delete it and the older MSBuild should work again.</p>
