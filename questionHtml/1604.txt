<td class="answercell">
    <div class="post-text" itemprop="text">
<p><a href="http://lucene.apache.org/nutch/">Nutch</a> is the best you can do when it comes to a free crawler.  It is built off of the concept of <a href="http://lucene.apache.org/java/docs/">Lucene</a> (in an enterprise scaled manner) and is supported by the <a href="http://hadoop.apache.org/core/">Hadoop</a> back end using <a href="http://labs.google.com/papers/mapreduce.html">MapReduce</a> (similar to Google) for large scale data querying.  Great products!  I am currently reading all about Hadoop in the new (not yet released) <a href="http://www.manning.com/lam/">Hadoop in Action</a> from manning.  If you go this route I suggest getting onto their technical review team to get an early copy of this title!  </p>

<p>These are all Java based.  If you are a .net guy (like me!!) then you might be more interested in <a href="http://incubator.apache.org/lucene.net/">Lucene.NET</a>, <a href="http://sourceforge.net/projects/dotnutch">Nutch.NET</a>, and <a href="http://code.google.com/p/hadoopdotnet/">Hadoop.NET</a> which are all class by class and api by api ports to C#.</p>
