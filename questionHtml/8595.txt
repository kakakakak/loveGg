<p>I downloaded the source code for Hadoop 2.2.0 and all of the relevant dependencies and built it/installed it/configured it. Note that I built it using MinGW from Git rather than Cygwin, if that matters. </p>

<p>I am running a job but the only output that I get is the input file. Logging from my mapper or reducer is not working, either - in fact, I don't even know how to get Hadoop to tell me if it is running my specific class implementation. </p>
