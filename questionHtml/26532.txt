<td class="answercell">
    <div class="post-text" itemprop="text">
<p>I also got these errors using cygwin 1.7:  </p>

<blockquote>
  <p>/usr/local/hadoop-0.21.0/bin/hadoop-config.sh: line 15: $'\r': command
  not found</p>
</blockquote>

<p>It appears that these files are not in DOS CR/LF format instead of the expected UNIX format.</p>

<p>You can confirm this with a text editor like Notepad++.  In the lower right hand part of the window, it can say Dos\Windows (very likely what you would see) or UNIX.</p>

<p>To work around these errors, in a cygwin window I used the command dos2unix *.sh in the bin folder and other folders (conf, etc...) as needed.</p>

<p>Hope this helps you!</p>
