<p>When you run a hadoop jar this is the command which you should run in the directory you put the jar in (e.g. /usr/lib/hadoop-0.20/bin )</p>

<ul>
<li>hadoop jar NAMEOFJAR.jar arg1 arg2 argN</li>
</ul>

<p>from your question this could be how to run it (make the cd to hadoop directory for the version of hadoop your running) </p>

<ul>
<li>cp MaxTemperature.jar /usr/lib/hadoop-0.20/bin</li>
<li>su hadoop</li>
<li>cd /usr/lib/hadoop-0.20/bin</li>
<li>hadoop jar MaxTemperature.jar input/ncdc/sample.txt output</li>
</ul>
